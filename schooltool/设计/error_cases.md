# 研知道 · 错误案例库

> **目的:** 用 12 个反面教材帮设计师 / AI 工具 / 守门人快速识别"什么不要做"
> **使用方式:** 与 design.md 配合使用,每张图发出前对照本库自检

---

## 为什么需要错误案例库

研知道的视觉系统是多轮设计取舍后才确立的。人脑学正面规则需要 5-10 次重复,学反面案例 1-2 次就记住。

错误案例库的真正作用:**把规范从"读"变成"看"**。当设计师/运营/AI 看到一组明确的"❌",规范的边界比正文长篇大论更清晰。

## 12 个核心错误

### ❌ 错误 01:用饱和紫蓝做主色

**症状:**
- 主色用 `#5B5FE8` 系饱和紫蓝
- 标题字用纯白 + 强对比
- 价格用 `#FFD700` 灯芒黄
- 添加"抢购!""爆款!""限时!"等促销词

**为什么错:**
这是**网红考研机构通用语言**(像某米考研、某途的视觉)。直接套用会让研知道丢失"不骗人真服务"的踏实感,**变成另一家高强度营销的考研机构**。

**怎么改:**
- 主色锁定赤土橙 #CF7140(品牌基因不动摇)
- 价格用主色 #CF7140 或主文字色 #2A251E,不用红/黄/紫蓝
- 营销文案改克制版:"2026 届冲刺专属价",不是"立即抢购"

**适用对照场景:** 淘宝主图、详情页、短视频引流图

---

### ❌ 错误 02:用党旗红 + 金黄星 + 印章感

**症状:**
- 用 `#C8312E` 鲜红色做主色
- 加 ★ 五角星
- "研知道"做成红底黄字的"印章"样式
- 配文用"权威认证""红色教育"

**为什么错:**
**致命错误。** 红色 + 黄色 + 五角星 + 印章 = "假冒官方机构",**有合规风险**(可能被认为冒充党政机关),且让客户觉得"不正经"——真正的体制内机构不会这样宣传自己。

**怎么改:**
- "党校"不需要视觉强调,文字直陈即可:"川渝党校 · MPA 双证"
- 主色保持赤土橙 #CF7140
- 永远不用红色作为主色或标识色

**适用对照场景:** 任何涉及"党校"业务的物料

---

### ❌ 错误 03:用英文做品牌识别

**症状:**
- 刊头用 `YANZHIDAO · EST. 2021`
- 文末用 `FILE NO. 042`
- 强调"国际感""文艺感"

**为什么错:**
研知道服务**川渝体制内中年人**——他们一辈子写中文、读中文、签中文档案。看到英文品牌识别会觉得"这家有点装"。**英文不等于高级,合适才高级**。

**怎么改:**
- 刊头用中文:`■ 研 知 道 · 成立于 2021 年 · 川渝在职考研`
- 刊尾用中文:`不骗人 · 真服务`
- **章节小标签可保留英文**(如 PRODUCT NOTE / CASE),但**品牌识别位必须中文**

**注意区分:** 章节标签 ≠ 品牌识别。前者是"内容分区",后者是"品牌签名"。

---

### ❌ 错误 04:拼图式产品堆叠 + 立体字

**症状:**
- 4 本教材旋转、倾斜、重叠摆放
- 标题用立体字 + 投影 + 描边
- 渐变色背景
- 红色"爆款"标签

**为什么错:**
这是**拼多多 / 微商详情页**的语法。强烈的视觉冲击伴随着**"廉价感"和"不可信感"**——目标人群(机关公务员、国企中层)对这种风格高度敏感,会条件反射觉得"骗钱"。

**怎么改:**
- 书本垂直整齐排列,**像图书馆陈列**
- 用扁平色块代替写实摄影(避免廉价摄影感)
- 无投影、无旋转、无渐变
- 信息密度通过"留白 + 大字 + 1px 黑线"传达

**核心原则:** 信息密度可以高,**视觉密度必须低**。

---

### ❌ 错误 05:用 emoji 和表情包

**症状:**
- 标题前后 🎉🔥💯
- 时间用 ⏰ 表示
- 落款 ✨🌟
- "今晚直播!不见不散!"

**为什么错:**
emoji = **小学生宣传海报 / 微商社群活动**。瞬间从专业服务机构变成业余团队。中年体制内用户**特别反感这种"表演热情"**。

**怎么改:**
- 所有 emoji 替换为:文字、数字、极简符号(`·` `—` `❶` `❷`)
- 时间直接写:`20:00 直播` 不写 `⏰ 20:00 直播 ⏰`
- 强调用赤土橙(主色)或加粗,不用 emoji

**特别注意:** 哪怕是社群消息、轻量提醒,也不用 emoji——研知道的"温度"靠文案传递,不靠表情包。

---

### ❌ 错误 06:海报场景用 SaaS 风圆角 + 大渐变

**症状:**
- 海报(朋友圈金句 / 公众号封面 / A4 招生海报)用 24px 圆角
- 大块 `linear-gradient` 渐变背景(紫蓝渐变 / 橙红渐变)
- 强阴影 `0 4px 16px rgba(0,0,0,0.08)`

**为什么错:**
这是**互联网 SaaS 产品风**搬到海报场景(Notion、Linear、Stripe 那种风格用错了地方)。海报需要的是"档案踏实感",SaaS 风暗示"轻盈、灵活、随时可改"——和"庄重、可信、长期"打架。

**关键区分**:
- 海报场景 → 直角档案 + 不用大渐变
- UI 场景 → 16-20px 圆角白卡 + 极轻阴影 + 极淡光晕(见错误 09)

**怎么改:**
- 海报主体一律直角(`border-radius: 0`)
- 海报内"卡片化模块"(如 A4 海报内的 CTA 卡 / 时间轴卡)可用 16px 圆角
- 海报极淡中心光晕可用(opacity ≤ 0.10),但**不允许大块 linear-gradient**
- 阴影最多 `0 2px 8px rgba(60,50,40,0.04)`(海报内卡片化模块可用)
- 永远不用渐变色做主元素(主标题 / 按钮 / 大色块)

**口诀:** 海报直角档案,UI 圆角白卡;光晕极淡,渐变不堆。

---

### ❌ 错误 07:文案用命令式 + 感叹号

**症状:**
- "亲!立即抢购!"
- "仅剩 3 名!不要错过!"
- "马上下单"
- 全篇感叹号
- 红色高亮"限时"

**为什么错:**
这是**微商话术 + 直播带货话术**。中年体制内用户会**条件反射性地反感**——这种话术暗示"对方在压迫你做决定",而体制内人最珍视的是"自己做决定"。

**怎么改:**
- 主语换成"我"(用户),不是"你/亲"(命令)
- ❌ "立即下单" → ✅ "我准备好了,加入 2026 届"
- ❌ "马上抢购" → ✅ "2026 届冲刺专属价"
- 全文不出现感叹号(陪伴式语气)
- 紧迫感来自"事实"(限 2026 届),不来自"催促"

---

### ❌ 错误 08:缺刊头刊尾(品牌基因丢失)

**症状:**
- 一张漂亮的金句卡,没有"研知道"标识
- 没有刊头(身份位缺失)
- 没有刊尾(主张缺失)
- 满屏只有内容,无品牌锚点

**为什么错:**
**最易被忽略的错误。** 漂亮的设计 + 没有品牌识别 = **"路边随便贴的纸条"**。客户看完不知道"这是谁说的",更不会记住"这是研知道"。

**怎么改:**
- **所有物料必有刊头**:`■ 研 知 道 · 成立于 2021 年 · 川渝在职考研`
- **所有物料必有刊尾**:`不骗人 · 真服务`
- 刊头 + 1px 黑线封口(顶部)
- 刊尾 + 1px 黑线封口(底部)

**这是研知道的"出生证"。** 漂亮的内容没有出生证,等于没出生。

## 错误 09-12(UI / 第二色 / token 一致 / 海报渐变)

> 这 4 个错误针对**UI vs 海报分类 / 主色 / 第二色 / 圆角分场景**规则下的常见踩雷。

### ❌ 错误 09:UI 场景用了硬直角(产品感缺失)

**症状:**
- 小程序首页用 0 圆角白卡(像 Excel 表格)
- 选项卡 / 单选 radio 用 0 圆角
- 卡片之间用 1px 黑线档案表分隔(海报风搬到 UI)
- CTA 是直角矩形(不是 pill)

**为什么错:**
"全场直角"在海报场景成立(档案踏实),**但在产品 UI 里会让用户觉得"工程师感太硬,不愿点"**。三顿半路径证明:UI 圆角白卡 + 极轻阴影 + 圆形 icon 头标,既保品味又有产品感。**规则是"分场景":UI 类允许 16-20px 圆角。**

**怎么改(标准):**
- 标准 UI 卡 `border-radius: 16px` + 极轻阴影 `0 2px 8px rgba(60,50,40,0.04)` + 0.5px 边框
- hero / 决策卡 `border-radius: 20px`
- 单选 radio `border-radius: 50%`(圆形)
- 圆形 icon 头标 `border-radius: 50%`,40-56px 直径
- CTA 始终 `border-radius: 999px`(pill 药丸)

**判定原则:** 给用户**点 / 选 / 滑**的 → 圆角白卡(UI 类) · 给用户**看 / 读**的 → 直角档案(海报类)

**适用对照场景:** 小程序 UI / 测评工具 / H5 落地页 / 淘宝详情页内嵌组件

---

### ❌ 错误 10:第二色绿用错语义

**症状:**
- 主标题某个关键词用 #5F8C6E 暖灰绿高亮(应该用主色赤土橙)
- CTA 按钮用绿色填充
- 错误/警告提示用绿色
- 卡片底色用浅绿(应该是白)
- "投入指数"进度条用绿色(应该是橙)

**为什么错:**
**第二色 #5F8C6E 暖灰绿是"积极/收益"语义专用色**——它在用户脑里建立"看到绿色 = 这是积极的"心智锚定。一旦绿色被滥用在非积极语义场景,这个心智锚定就崩塌,品牌识别度归零。

**第二色铁律(必背):**

```
✅ 决策报告"收益指数"进度条 + 标签
✅ "核心收益"小卡 icon + 标题
✅ 时间轴"已完成节点"
✅ 案例列表"上岸"标签
✅ 元数据栏"LEARNERS · 1,100+"等积极数据高亮

❌ 主标题 / 正文 / 卡片底色 / 边框
❌ CTA 按钮(CTA 永远赤土橙)
❌ 关键词高亮(关键词永远赤土橙)
❌ 警告 / 错误 / 中性提示
❌ "投入"语义(投入永远用赤土橙)
```

**怎么改:**
检查每一处绿色 #5F8C6E 出现,问自己:"这里是'积极/收益/上岸'语义吗?"——不是 → 全部改回主色赤土橙 #CF7140。

**适用对照场景:** 决策报告 / 案例列表 / 数据可视化 / 任何使用了第二色的物料

### ❌ 错误 11:UI 和海报视觉系统脱钩(token 不一致)

**症状:**
- 小程序 UI 用赤土橙 #CF7140,A4 海报却用旧主色 #7A4A2B(没同步)
- UI 用衬线 hero,海报却用全 sans
- UI 字体是思源宋体/黑体,海报临时换了"霞鹜文楷"或其他字体
- 两个物料放在一起,看不出是同一品牌

**为什么错:**
第 17 章「UI vs 海报统一视觉系统」的核心是**"视觉 token 完全统一 / 表达类型分两类差异化"**。token 不统一 = 系统失效 = 品牌识别度归零。**差异化不等于脱钩**——UI 和海报必须共用相同的色/字/CTA/刊尾,只在布局/构图/信息密度上差异化。

**怎么改(强制 token 一致清单):**

```
所有 UI 和海报必须共用:
✅ 主色 #CF7140 + 第二色 #5F8C6E
✅ 米白底 #F5EFE7 + 卡片白 #FFFFFF
✅ 字体策略:衬线 hero + sans UI / 正文
✅ 中英文半角空格全场化
✅ CTA 永远 pill 药丸 + 主色填充 + 白字 + 右箭头
✅ 海报必带"不骗人 · 真服务"刊尾(UI 可省)

差异化允许的:
⚡ 圆角(UI 16-20px / 海报直角档案)
⚡ 主标字号(UI 24-32px / 海报 60-116px)
⚡ 阴影(UI 极轻 / 海报默认无)
⚡ 渐变(UI 无 / 海报极淡光晕 ≤ 0.10)
```

**自检方法:** 把同一品牌的 UI 和海报放并排,问"看起来像同一个品牌做的吗?"——答案必须是"是"。

**适用对照场景:** 跨场景的物料生产(UI + 海报混合发布的活动)

### ❌ 错误 12:海报场景大渐变 / 满底彩色 / 装饰几何元素

**症状:**
- 海报背景用 `linear-gradient(135deg, #CF7140, #5F8C6E)` 主色到第二色大渐变
- 满底纯色块(非 L1 短视频封面场景)
- 海报加装饰性几何元素(背景几何图、装饰线、彩色色块)
- 引入紫色 / 蓝色 / 黑色 第三色装饰

**为什么错:**
渐变规则虽然**允许极淡光晕**(opacity ≤ 0.10 的 radial-gradient),但**不允许大块 linear-gradient**——线性渐变会让画面变成"装饰品",失去研知道"档案克制"的核心气质。**只有 L1 短视频封面允许主色满底反白**(强冲击力场景的例外),其他海报场景必须米白底。

**怎么改:**

```
✅ 允许:
- radial-gradient(circle, rgba(207,113,64,0.08) 0%, transparent 70%) 极淡光晕
- L1 短视频封面 主色满底 + 米白字反白(强冲击场)
- 海报内 1-2 个卡片化模块用极轻阴影

❌ 禁用:
- 任何 linear-gradient(包括主色到第二色 / 主色明度渐变 / 主色到米白渐变)
- 满底纯色块(L1 短视频反白除外)
- 装饰性几何元素(几何图 / 装饰线 / 彩色色块)
- 第三 accent 色(蓝 / 紫 / 任何冷色)
```

**判定:** 渐变是为了"氛围"(极淡光晕)合规 / 渐变是为了"装饰"(linear-gradient 主元素)禁。

**适用对照场景:** 朋友圈金句 / 公众号封面 / A4 招生海报 / 抖音封面 / 小红书图文

## 12 个错误的共同规律

**所有错误的根源只有一句话:**

> **把研知道做成了"别人"——网红机构、电商详情、微商朋友圈、政府官网、SaaS 产品。**
> **或者把"对的别人"用错了场景——SaaS 风搬到海报、海报硬直角搬到 UI。**

研知道是研知道:
- 米白底 #F5EFE7 + 赤土橙 #CF7140 + 暖灰绿 #5F8C6E(收益专用)
- 中文刊头 + 中文刊尾(「不骗人 · 真服务」)
- 思源衬线 hero + 思源黑体 UI
- **海报直角档案 / UI 圆角白卡**(分场景关键)

**这套语言长得不像任何"现成模板",这就是它最大的识别度。**

## 自检流程(给守门人和设计师)

每张图发出前,**至少过一遍 12 个错误**:

```
基础 8 条:
☐ 主色是不是 #CF7140?(防错误 01、02)
☐ 刊头是不是中文?(防错误 03)
☐ 没有立体字、拼贴、大渐变?(防错误 04、06)
☐ 没有 emoji?(防错误 05)
☐ 文案没用命令式 + 感叹号?(防错误 07)
☐ 海报有刊头 + 刊尾?(防错误 08)

补充 4 条:
☐ 这是 UI 还是海报?圆角对吗?(防错误 09 · UI 16-20px / 海报直角)
☐ 第二色 #5F8C6E 只用在"积极/收益"场景了吗?(防错误 10)
☐ UI 和海报视觉 token 一致吗?(防错误 11)
☐ 海报没有大块 linear-gradient / 装饰几何 / 第三色吗?(防错误 12)
```

**12 项全过 → 可发**
**任何一项不过 → 返工**

## 错误案例库的扩展机制

随着实战推进,**会遇到 design.md 没覆盖的边界情况**。守门人(ww)发现新错误时,按以下格式补充:

```
### ❌ 错误 0X:[错误名]
症状:[列举具体表现]
为什么错:[根源分析]
怎么改:[修正方法]
适用对照场景:[哪些物料类型容易犯]
```

**目标:** 案例库每月扩充 2-3 条,3 个月后形成 15-20 条的成熟错误库——**这才是真正的规范护城河**。

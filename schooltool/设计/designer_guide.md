## 一 · 5 分钟理解研知道是谁

### 1.1 业务事实

```
品牌名:研知道
成立:2021 年(已 4 年)
业务:川渝在职考研培训
核心服务:党校在职研究生 + 199 管理类联考(MPA/MBA/MEM)
客户群:川渝体制内 30-40 岁(公务员/事业单位/国企中层/乡镇基层)
团队规模:约 10 人
累积学员:1,100+ 人
```

### 1.2 客户画像(决定视觉口味)

**典型客户长这样:**
- 35 岁市直机关科员,想晋升副科,需要学历
- 32 岁国企中层,职务调整在即,想读个 MPA
- 38 岁乡镇科员,有职级提升空间,想拿党校学历
- 31 岁事业编人员,生完二胎想读个非全日制

**他们的视觉环境:**
- 每天打交道:政府文件、报告、PPT、Excel、微信群消息
- 不读:小红书时尚、独立设计杂志、品牌艺术海报
- 平时浏览:抖音短视频、央视新闻、本地公众号、拼多多/淘宝

### 1.3 核心商业判断

**研知道的差异化是什么?**

> **"不骗人 · 真服务"** —— 在一片浮夸的考研机构里,做"老老实实办事"的那家。

这一句话决定了**所有视觉决策**:
- 不能像网红机构(那是"看起来厉害"路线)
- 不能像律所政府(那是"我审视你"路线)
- 不能像度假酒店(那是"生活方式"路线)
- 应该像 **专业服务机构 + 编辑部**(踏实可信 + 懂你温度)

## 二 · 视觉基因:5 件不动的事

研知道的"DNA"由 5 个元素构成。**这 5 件事在任何物料上都必须出现,缺一不可**:

### 2.1 赤土橙主色 #CF7140

```
为什么是这个色:
- 不像党旗红(机构感太强,合规风险)、也不像陶土橙(太柔)、也不像墨绿(度假酒店)
- 介于"温暖"和"专业"之间,像 Claude.ai / 三顿半早期那种"高级暖橙"
- 保留"档案温度"的同时具备"产品感"

用法:
- CTA 按钮:赤土橙底 + 白字
- 进度条投入态:赤土橙填充
- 关键词高亮:在大标题里强调某个词
- 小方块签名:8px 赤土橙小方块作为品牌锚
- 浅赤土 #F1E0D3:chip 浅底、icon 圆背景、卡内浅底
- 主色透明 rgba(207,113,64,0.08):极淡光晕、hover 背景

衍生色阶(供进阶用):
- Primary       #CF7140  主色(70-80% 占比)
- Primary-soft  #F1E0D3  浅赤土(chip / icon 圆背景)
- Primary-dim   rgba(207,113,64,0.08)  极淡(光晕 / hover)
```

### 2.1.5 第二色 · 暖灰绿 #5F8C6E

```
为什么需要第二色:
- 长期物料产出会"千图一色",失去新鲜感
- 研知道不能引入冷色(蓝/紫),会破坏暖色系档案克制气质
- 暖灰绿 #5F8C6E 与赤土橙同处暖色家族,饱和度 ≤ 60,在米白底有可读性
- 它承担明确语义:"积极 / 收益 / 上岸"

严格使用规则(铁律):
- ✅ 决策报告的"收益指数"进度条
- ✅ 决策报告"核心收益"小卡(icon + 标题)
- ✅ 时间轴中"已完成节点"的颜色变体
- ✅ 案例列表中"上岸"标签 chip
- ✅ 元数据栏中"LEARNERS · 1,100+"等积极数据高亮
- ❌ 主标题 / 正文 / 卡片底色 / 边框
- ❌ CTA 按钮(CTA 永远赤土橙)
- ❌ 任何"非积极/收益"语义场景

浅色:暖灰绿浅 #DAE5DC,用法等同浅赤土
```

### 2.2 8px 主色小方块

```
■ 研 知 道

为什么:
- 它就是研知道的"印"——像古书扉页的藏书章
- 8px 是肉眼舒适大小:再小看不见,再大喧宾夺主(实操范本:品牌视觉5件基因全景图)
- 出现在所有刊头/刊尾/Logo 旁

绝对禁止:
- 不要换成圆点 ●(质感软)
- 不要换成五角星 ★(政府感)
- 不要加边框(失去印章感)
```

### 2.3 中文刊头(身份)

```
■ 研 知 道                成立于 2021 年 · 川渝在职考研

为什么:
- 中文是研知道的母语,也是用户的母语
- 不用 YANZHIDAO / EST. 2021 (英文标识 = "装外国人")
- 章节标签可英文(PRODUCT NOTE),但品牌识别必须中文
- "研 知 道" 三字加宽字距 4px,像印章篆刻
```

### 2.4 中文刊尾(主张)

```
不骗人 · 真服务

为什么:
- 这是研知道的"商业道德可视化"
- 出现在每张物料底部,1px 黑线封口
- 思源宋体 SemiBold,不浮夸
- 与刊头(身份)分两端,互相强化
```

### 2.5 边框美学 · 分场景

```
KV / 海报类(用户在 feed 流里看):
- 直角 + 0.5px / 1px 黑线档案表
- 朋友圈金句 / A4 招生海报 / 公众号封面 / 短视频封面 默认走这套
- 0.5px 边框比 1px 更"贵",看起来薄但有质感

UI 类(用户在手机里完成任务):
- 16-20px 圆角白卡(标准卡 16px / hero 卡 20px)
- 极轻阴影 0 2px 8px rgba(60,50,40,0.04) + 0.5px Border 边框 配合分层
- 圆形 icon 头标 / 单选 radio / 圆形进度 dot 也用 50% 圆角
- 小程序 / H5 测评工具 / 淘宝详情页内嵌组件 走这套

为什么这样分:
- 全场直角让产品 UI 用户觉得"工程师感太硬,不愿点"
- 三顿半路径证明:UI 圆角 16-20px + 极轻阴影,既保品味又有产品感
- KV/海报保留直角,因为海报场景靠"档案感"建立可信度,圆角会削弱

唯一全场通用:
- CTA 按钮始终 999px 药丸圆角(任何场景)
- chip 标签始终 999px 圆角(任何场景)
```

**判定快查**:
- 用户要**点/选/滑** → UI 类 → 圆角白卡
- 用户**只是看一眼** → 海报类 → 直角档案表

## 三 · 三层受众,三种"音量"

研知道的客户在不同阶段需要不同的"视觉音量"。**同一个品牌,要会切换"说话方式"**:

### 3.1 L1 · 冷启动(陌生人,需要立刻吸引)

**场景:** 短视频引流图、第一次曝光、社群广撒图、淘宝搜索结果展示
**心态:** 用户 1 秒决定看不看

**视觉策略:**
- **大色块 + 反白**:赤土橙底 + 白字
- **思源黑体 Heavy**(字重 900,有冲击力)
- **直白疑问句**:"科级晋升前,要不要先把学历补上?"
- **大数字 / 大价格**:¥499 用 Inter 数字 + 字重 900
- **CTA 醒目**:白底 + 主色字 + 字重 700

**禁忌:**
- 不要衬线宋体(L1 用户没耐心读宋体)
- 不要 P.S. 后记(没空间)
- 不要复杂版式(只有 1 个核心信息)

**典型场景:** 淘宝主图、抖音封面、朋友圈拉新图

---

### 3.2 L2 · 半信任(听过没买,在比较)

**场景:** 详情页前 3 屏、淘宝搜索点进去后的第一屏、公众号头条
**心态:** 用户 3-5 秒决定继续看不看

**视觉策略:**
- **米白底 + 白卡片**(不再用反白大色块)
- **思源黑体 Heavy**(还是黑体,但环境是米白)
- **大字号标题**(34-42px),关键词主色高亮
- **强价格锚点**:白底 + 1px 黑边 + 思源黑体 Heavy 32px 价格
- **保留营销三件套**:限 / 包邮 / 7 天无忧

**关键平衡:** 既要转化(L1 的功能),又要专业(L3 的气质)

**典型场景:** 详情页 Hero、小红书封面、淘宝店铺 banner

---

### 3.3 L3 · 信任(已决定了解,愿意慢下来读)

**场景:** 小程序内页、详情页主体、私域配图、编辑部内容
**心态:** 用户愿意读,愿意被说服

**视觉策略:**
- **米白底 + 白卡片**
- **思源宋体 SemiBold**(衬线!这是 L3 的核心标志)
- **章节小标签** + 中文大标题反差
- **36×2px 主色短横线**(签名底下的横线)
- **正文 14px / 行高 1.95**(让眼睛慢下来)
- **引句区:赤土橙背景 + 白字**(态度宣告)
- **P.S. 后记:温度的核心**(用"我们知道你..."开头)

**典型场景:** 小程序首页、详情页主体、朋友圈金句、公众号内文

---

### 3.4 三层选择速查表

```
看见一个需求 → 问自己:用户处于哪个阶段?

陌生人,要点击      → L1
听过没买,在比较    → L2
已决定了解,愿意读  → L3
```

## 四 · 字体系统:为什么是这个组合

### 4.1 双字体的逻辑

```
思源黑体 Heavy(L1/L2)→ 力量感、直白、中低学历秒懂
思源宋体 SemiBold(L3)→ 文化感、深度、有耐心读
```

**为什么不只用一个?**
- 单纯黑体 → 像考研机构(嘈杂)
- 单纯宋体 → 像三联生活周刊(曲高和寡)
- **黑体打前阵(L1/L2 抓眼球),宋体定底色(L3 留住人)**

### 4.2 思源系字体的优势

```
1. 完全免费可商用(Apache 2.0 / SIL OFL)
2. 无版权风险
3. 简体中文优化最好(比谷歌 Noto 更适合中文)
4. 字重齐全(从 ExtraLight 到 Heavy 都有)
5. Adobe + Google 联合开发,长期维护
```

**下载地址:**
- 思源黑体:https://github.com/adobe-fonts/source-han-sans
- 思源宋体:https://github.com/adobe-fonts/source-han-serif

### 4.3 字号阶梯参考

```
L1/L2(黑体 Heavy):
  Display-XL:  44-56px(短视频封面、Hero 主标)
  Display-L:   34-42px(L2 详情页主标)
  Display-M:   28-32px(L1 副标题)

L3(宋体 SemiBold):
  H1:  28-32px(L3 页面级主标)
  H2:  20-24px(章节标题、引句)
  H3:  18px(卡片标题)

通用:
  Body:     14px / 行高 1.95(L3 正文)
  Caption:  13px(说明)
  Tag:      11-12px(小标签)
  Tiny:     10-11px(刊头刊尾品牌标识)
```

## 五 · 排版核心法则:让规范"看得见"

### 5.1 三明治结构(L3 物料的标准结构)

```
┌─────────────────────────────────────┐
│ ■ 研 知 道  ·  成立于 2021 年  · 川渝在职考研 │  ← 刊头 = 身份
├─────────────────────────────────────┤
│                                     │
│ [PRODUCT NOTE]                      │  ← 英文章节小标签
│                                     │
│ 最后一个月,                          │  ← 衬线宋体大标题
│ 选这两本就够了。                     │
│                                     │
│ ─────                               │  ← 36×2px 主色短横线
│                                     │
│ 写给 2026 届四川党校在职考研同学      │  ← 副标
│                                     │
│ [正文 ……]                            │
│                                     │
│ [引句区:主色底 白字]                 │
│                                     │
│ [P.S.] 我们知道在职备考很辛苦……      │  ← 温度
│                                     │
├─────────────────────────────────────┤
│ 不骗人 · 真服务         第 042 号     │  ← 刊尾 = 主张
└─────────────────────────────────────┘
```

### 5.2 章节小标签法

每个内容块开头都用**英文小标签 + 中文大标题反差**:

```
[REGIONAL FOCUS]    ← 11px / 字距 3-4px / 主色
精准匹配川渝院校    ← 22px / 衬线 / 黑色
```

**为什么这样做:**
- 英文小标签像杂志的"页眉" → 给版面增加层次
- 中文大标题承担实际信息 → 给眼睛重音
- 两者反差大,视觉张力强

**可用英文章节标签:** `PRODUCT NOTE` / `CASE` / `CONTENTS` / `P.S.` / `NOTICE` / `PROFILE`

**注意:** 这是**章节标签**(给内容分区用),不是**品牌识别**。品牌识别必须中文。

### 5.3 引句区:研知道的"态度宣告"

```
背景:赤土橙 #CF7140
文字:白色,思源宋体 Medium 18-20px
装饰:超大引号(48px,白色 40% 透明度)
署名:— 研知道·XX(11px / 字距 3px / 白色 60% 透明度)

举例:
┌─────────────────────────┐
│ "                      │
│  先背手册,再刷题。      │
│  简单,但是有效。        │
│                         │
│  — 研知道·33 法         │
└─────────────────────────┘
```

**为什么赤土橙背景:** 在米白主调中,主色块就是品牌"重音"——它把"研知道说什么"用最强的方式标记出来。

### 5.4 表格序号档案法

```
─────────────────  ← 1px 黑线起
01  内容名称           备注
─ ─ ─ ─ ─ ─ ─ ─ ─  ← 0.5px 灰线分隔
02  内容名称           备注
─ ─ ─ ─ ─ ─ ─ ─ ─
03  内容名称           备注
─────────────────  ← 1px 黑线收
```

**关键细节:**
- 01/02/03 用思源宋体 SemiBold,颜色三级文字 #8A8175
- 主文用思源宋体 Medium,颜色墨黑
- **序号"低调",主文"突出"**——形成视觉重音
- 上下 1px 黑线封口,中间 0.5px 灰线分隔

### 5.5 P.S. 后记法(温度的核心)

每张关键 L3 物料都有一段 P.S.:

```
[P.S.]                                    ← 11px / 字距 3px / 主色

我们知道在职备考很辛苦——白天上班、晚上学习,
周末还要陪家人。这套资料的目标不是让你"学得多",
是让你"学得对"。剩下的时间,留给生活。

— 研知道·编辑部                            ← 右对齐
```

**为什么这样写:**
- "我们知道你..."的句式 = **承认用户的处境**(共情而不煽情)
- 提供"反共识"建议(不是学得多,是学得对)
- 给生活留位置(不是只谈学习)

**这一段是研知道作为"服务机构而非营销机构"的核心证据。**

## 六 · KV 系统:从母版到衍生

### 6.1 KV 系统总览

研知道 KV 系统由**两层组成**:

```
1. 母版 KV(4 张):定义品牌"基因",作为衍生的源头
   - M01: 1:1 方版母版    1080×1080
   - M02: 3:4 竖版母版    750×1000
   - M03: 2.35:1 横版母版 900×383
   - M04: 9:16 超竖版母版 1080×1920

2. 场景衍生 KV(N 张):母版在具体场景的"翻译"
   - D01: 淘宝 1:1 主图(L1)
   - D02: 朋友圈金句图(L3)
   - D03: 公众号头条封面(L3)
   - D04: 交付群通知图(L3)
   - D05: 小红书竖版封面(L2)
   - ……(按 12 场景扩展)
```

### 6.2 派生 5 步法

设计师拿到一个新需求,按这 5 步走:

```
1️⃣ 看场景 → 选比例
   朋友圈/淘宝主图 → 1:1 → 用 M01 母版
   通知/小红书 → 3:4 → 用 M02 母版
   公众号头条/banner → 2.35:1 → 用 M03 母版
   抖音/视频号 → 9:16 → 用 M04 母版

2️⃣ 看层级 → 选字体
   L1 → 思源黑体 Heavy
   L2 → 思源黑体 Heavy
   L3 → 思源宋体 SemiBold

3️⃣ 装刊头 + 刊尾(必做)
   尺寸 < 1080×1500 → 刊头 A 式(左品牌右信息)
   尺寸 ≥ 1080×1500 → 刊头 C 式(双行堆叠)
   刊尾统一:不骗人 · 真服务

4️⃣ 替换主标题 → 写场景文案

5️⃣ 过 9 步检查清单(见 design.md 第九章)
```

### 6.3 派生时"什么必须保留 / 什么可以变"

```
【必须保留】(品牌基因 5 件)
✓ 赤土橙 #CF7140 主色
✓ 8px 主色小方块签名
✓ 中文刊头:研 知 道 + 成立于 2021 年 + 川渝在职考研
✓ 中文刊尾:不骗人 · 真服务
✓ 1px / 0.5px 档案线边框美学(KV/海报直角档案;UI 场景允许圆角白卡,见 design.md §17)

【允许变化】(场景适配 5 维度)
○ 底色:米白 / 主色反白(L1 转化场常用反白)
○ 字体:L1/L2 黑体 ↔ L3 宋体
○ 主标题:可换金句/标题/价格
○ 布局:横/竖/方/超竖
○ 信息密度:L1 少 → L3 多
```

## 七 · 12 场景速查

## 九 · 设计师工作流(从需求到交付)

### 9.1 标准流程

```
[1] 接需求 (15 分钟)
    ├─ 这是哪个场景?(查 12 场景表)
    ├─ 这是什么层级?(L1/L2/L3)
    ├─ 用什么尺寸?(查比例)
    └─ 谁是受众?(陌生人/比较中/已决定)

[2] 确定母版 (5 分钟)
    └─ 4 张比例母版选 1 张作起点

[3] 选刊头 (1 分钟)
    ├─ 尺寸 < 1080×1500 → A 式(左品牌右信息)
    └─ 尺寸 ≥ 1080×1500 → C 式(双行堆叠)

[4] 写文案 (15-30 分钟)
    ├─ 主标题:对话语言(参考 8.2)
    ├─ 副标:场景说明
    ├─ 引句(L3):态度宣告
    └─ P.S.(L3):温度后记

[5] 排版 (30-60 分钟)
    ├─ 三明治结构(刊头 - 主体 - 刊尾)
    ├─ 章节小标签 + 中文大标题反差
    ├─ 36×2px 主色短横线锚点
    └─ 表格用档案法(01/02/03 + 1px 黑线)

[6] 自检 (10 分钟)
    └─ 过 9 步检查清单(见 design.md 第九章)
       + 错误案例对照(见 error_cases.md)

[7] 提交守门人审核
    └─ 守门人(ww)对照检查清单审核

[8] 通过 → 归档到视觉档案库
```

### 9.2 时间预估

```
首次上手:1.5-2 小时 / 张(熟悉规范)
熟练后:30-60 分钟 / 张(L3 内容物料)
        15-30 分钟 / 张(L1/L2 套模板)
```

### 9.3 守门人审核机制

每张图发出前,守门人(ww)对照 9 步检查清单审核:

```
1. ❓ 判定层级了吗?      L1 / L2 / L3 必须明确
2. ❓ 底色对吗?          米白 / 纯白 / 赤土橙 三选一
3. ❓ 主色对吗?          必须是赤土橙 #CF7140
4. ❓ 字体对吗?          L1/L2 黑体 Heavy,L3 宋体 SemiBold
5. ❓ 圆角对吗?          UI 类 14-24px 圆角白卡(默认 18) / KV 海报直角档案 / CTA 永远 pill
6. ❓ 刊头对吗?          中文 + 小尺寸 A / 大尺寸 C / 轻量可简
7. ❓ 刊尾对吗?          "不骗人 · 真服务" + 1px 黑线封口
8. ❓ 气质对吗?          "档案克制 + 编辑部温度"
9. ❓ 不像 AI 出的?      过 design.md §13 去 AI 自检 5 项
```

任何一项不过 → 返工。

## 十 · 给设计师的几句心里话

### 10.1 关于"克制"

研知道的设计语言看起来"简单",但**简单不等于好做**。

- 简单 = 每一个元素都有理由
- 好看 = 多个元素堆叠覆盖了"没理由"

**做研知道的设计,你做减法的能力比做加法的能力重要。**

### 10.2 关于"边界"

design.md 不可能覆盖所有情况。**你会遇到规范没写的场景**——这时候问自己 3 个问题:

```
1. 这个决定让"研知道更像研知道"还是"更像别的什么"?
2. 中年体制内人看见这个,会不会觉得"这家有点装"?
3. 守门人(ww)看见会皱眉头吗?
```

**3 个问题都过 → 可能对了。**

### 10.3 关于和 AI 工具配合

研知道的设计**不是设计师 vs AI 二选一**,是**设计师 + AI 共同生产**:

- AI 出底稿(快、规范、不偏)
- 设计师定稿(品味、细节、审美)
- 守门人把关(基因、统一、不飘)

**你的价值:** 不是和 AI 比谁画得快,是**判断 AI 出的稿哪里"有点不对"**。

### 10.4 关于品味

研知道的视觉气质不是"小众""高级""文艺",是:

> **像一位认真做事的中年人——不卑不亢,不慌不忙,做出来的东西经得起看。**

如果你的设计直觉总是想"加点装饰""做得再活泼一点""加点温度",**克制住**。

研知道的温度不在视觉里,**在文案里、在 P.S. 里、在"我们知道你..."这句话里。**

视觉负责"骨头硬",文字负责"血液暖"。

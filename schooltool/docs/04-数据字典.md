# 选校诊断工具 · 数据字典

> 版本：v0.1 | 日期：2026-05-05
> 用途：作为前端开发、后端开发、内容录入、推荐逻辑联调的统一字段基线
> 前置：`02-产品需求文档-V1.md`、`专项/01-诊断逻辑与推荐引擎.md`

---

## 一、文档目的

本文件不讨论页面长什么样，也不讨论推荐结论本身是否合理。

本文件只解决一件事：

**把系统里会出现的核心对象和字段先统一。**

只有字段统一，后续这三件事才能并行推进：

- 前端知道每个页面渲染什么
- 后端知道接口和数据表要提供什么
- 内容侧知道每条信息该补到哪个字段里

---

## 二、数据对象总览

V1 先统一以下 8 类核心对象：

1. 用户画像 `user_profile`
2. 诊断题 `question`
3. 诊断答案 `answer_option`
4. 路径方案 `path_option`
5. 院校项目 `school_program`
6. 推荐结果 `recommendation_result`
7. 案例内容 `case_story`
8. 内容来源 `source_reference`

---

## 三、对象字段定义

## 3.1 用户画像 `user_profile`

用于承接答题数据、生成推荐结果、沉淀顾问沟通基础信息。

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 用户画像唯一标识 |
| region | enum | 是 | 地区：四川 / 重庆 / 其他 |
| system_type | enum | 是 | 系统类型：公务员 / 参公 / 事业编 / 教育医疗 / 国企银行 / 乡镇街道 |
| job_track | string | 否 | 具体岗位方向 |
| education_level | enum | 是 | 当前学历：大专 / 全日制本科 / 非全本科 / 研究生 |
| party_member | boolean | 否 | 是否党员 |
| age_range | enum | 是 | 年龄段：25以下 / 26-30 / 31-35 / 36-40 / 41以上 |
| goal_type | enum | 是 | 目标类型：晋升 / 涨薪 / 遴选 / 调任 / 防御性学历 / 解决学历短板 / 其他 |
| budget_range | enum | 否 | 预算：2万以下 / 2-5万 / 5-10万 / 10万以上 |
| exam_tolerance | enum | 否 | 对考试难度接受度：低 / 中 / 高 |
| preferred_study_mode | enum | 否 | 学习方式偏好：周末 / 集中授课 / 线上为主 / 无明显偏好 |
| notes | text | 否 | 顾问备注或补充信息 |

## 3.2 诊断题 `question`

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 题目唯一标识，如 `Q1` |
| code | string | 是 | 字段映射代码，如 `region` |
| title | string | 是 | 题目标题 |
| subtitle | string | 否 | 题目补充说明 |
| question_type | enum | 是 | 单选 / 多选 / 条件题 / 阻断题 |
| order_index | number | 是 | 题目顺序 |
| is_active | boolean | 是 | 是否启用 |
| visible_condition | json | 否 | 出题条件 |

## 3.3 诊断答案 `answer_option`

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 答案唯一标识 |
| question_id | string | 是 | 所属题目 |
| label | string | 是 | 展示文案 |
| value | string | 是 | 实际存值 |
| score_tags | json | 否 | 用于规则引擎打标签 |
| next_question_id | string | 否 | 条件跳题用 |
| is_blocking | boolean | 否 | 是否触发阻断提示 |
| blocking_message | text | 否 | 阻断提示文案 |

## 3.4 路径方案 `path_option`

用于表达“党校 / MPA / MBA / 教育硕士 / 先升本再读研”等路径级对象。

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 路径唯一标识 |
| name | string | 是 | 路径名称 |
| category | enum | 是 | 党校 / 非全专硕 / 学历过渡 / 暂缓决策 |
| summary | text | 是 | 路径一句话说明 |
| suitable_for | text | 是 | 适合人群 |
| main_benefit | text | 是 | 核心收益 |
| main_risk | text | 否 | 主要风险 |
| time_cost | string | 否 | 时间成本 |
| money_cost | string | 否 | 费用区间 |
| difficulty_level | enum | 否 | 低 / 中 / 高 |
| recognition_scope | text | 否 | 认可范围说明 |
| status | enum | 是 | 启用 / 备用 / 暂停 |

| learning_mode | string | 否 | 授课方式 |
| tuition_range | string | 否 | 学费区间 |
| schooling_years | string | 否 | 学制 |
| exam_subjects | string | 否 | 考试科目 |
| difficulty_tag | enum | 否 | 低 / 中 / 高 |

## 五、字段治理规则

1. 新增字段前，先判断是否能复用现有字段。
2. 同一个概念只允许一个主字段，不允许前端和后端各自起名。
3. 所有“待核实”信息必须落到 `source_reference` 或 `last_verified_at`。
4. 与推荐结果直接相关的字段，不得只写在文案里而不入结构化字段。

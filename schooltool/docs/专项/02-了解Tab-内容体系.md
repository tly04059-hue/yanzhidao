## 一、"了解"Tab 在产品中的角色

### 为什么需要这个 Tab

原产品设计假设用户"已决定读研，只是不知道选什么"，直接从答题开始。但实际情况是：

- 很多用户还没决定要不要读研
- 很多用户不清楚学历在自己系统里到底有什么用
- 很多用户是拼凑出的半正确结论来找研知道（比如"我听说四川党校好考"）
- 很多用户只是想收集信息，还没准备答题

"了解"Tab 就是接住这批用户的**浏览式入口** — 他们可以不答题，直接看内容，看完自己判断。

### 和"测一测"Tab 的关系

| 维度 | 了解 Tab | 测一测 Tab |
|------|---------|----------|
| 交互 | 浏览式 | 答题式 |
| 用户状态 | 犹豫 / 收集信息 | 已决定要判断 |
| 输出 | 结构化公开信息 | 个性化推荐 |
| 用时 | 2-10 分钟（可停可走） | 3 分钟（必须完成） |
| 留存 | 高（可反复看） | 一次性 |
| 转化 | 浏览详情页底部 CTA → 导流到测一测 | 结果页 CTA → 留资 |

两者是**平行功能**，不是前后关系。用户可以：
- 只用了解 Tab（信息收集型）
- 只用测一测 Tab（已有意向型）
- 先用了解再用测一测（模糊期走向决策期）

## 二、内容原则（硬约束）

### 只做两件事

1. **政策解释**（公开信息 + 依据来源 + 积极性导向）
2. **案例匹配**（基于真实学员数据）

### 不做的事

- 不做鸡汤（"改变命运"、"奋斗逆袭"、"学历是你的武器"…）
- 不做抽象激励（"相信自己"、"未来可期"…）
- 不做主观评价（"党校其实挺好的"、"MPA 才是正确选择"…）
- 不做未标来源的数据（所有数字必须有出处）
- 不做人生建议（"你应该考虑一下家庭"…）

### 积极性导向怎么理解

政策本身是中性的，但解读时选择"这项政策对你意味着什么机会"的角度，而不是"你如果不做会怎样"的恐惧角度。

**正例：**
> 2019 年中办《公务员职务与职级并行规定》出台后，研究生学历在职级并行起始档位上有明确加分。对你来说，这意味着同龄同入职的本科同事晋升到一级科员用 4 年，研究生学历的你用 3 年。

**反例（避免）：**
> 如果你不读研，你将被同事甩开，一辈子卡在科员。

### 所有内容的标准结构

任何一段内容都要包含以下三个组件：

```
【政策依据】：文件名 + 年份 + 关键条款
【实际作用】：对用户的具体影响（可量化更好）
【学员数据】：真实统计或案例（标注样本量）
```

缺任何一个都不能上线。

## 三、四种浏览维度

### 维度 1：按系统

**分类选项**（对应 Q1）：

| 分类 | 覆盖岗位 |
|------|---------|
| 党政机关 | 组织/宣传/纪委/财政/发改/经信等 |
| 公检法纪检 | 公安/法院/检察院/纪委监委 |
| 乡镇街道 | 社区/村镇/基层党建/民政 |
| 教育医疗 | 教师/医护/学校医院行政 |
| 国企银行 | 国企管理岗/技术岗/银行/城投 |
| 民族地区 | 甘孜/阿坝/凉山 |

**每个系统的内容侧重：**

- 该系统的**晋升机制**（职级并行怎么走、竞争上岗看什么）
- 该系统的**学历作用**（什么场合硬要、什么场合加分）
- 该系统的**遴选/转岗政策**（能不能出来、要什么学历）
- 该系统**学员的典型选择**（基于样本的统计分布）
- 该系统的**推荐路径**（首选+备选）

**示例：财政系统**（完整内容见第六节）

### 维度 2：按目标

**分类选项**（对应 Q6）：

| 分类 | 描述 |
|------|-----|
| 本单位晋升 | 职级并行 / 竞争上岗 |
| 想遴选 | 到上级机关（省直/市直/中央） |
| 转公务员 | 从国企 / 事业编转公务员 |
| 职称评审 | 教育医疗 / 事业编专技岗 |
| 防御性学历 | 不想被比下去，先拿着 |

**每个目标的内容侧重：**

- 该目标的**硬政策要求**（学历是不是硬门槛）
- **单证 vs 双证**的差异（党校 vs MPA/MBA）
- 该目标对应的**时间窗口和年龄限制**
- **哪些路径能实现这个目标、哪些不能**
- 有这个目标的**学员怎么选的**

**示例标题：**
- "想遴选？你必须知道的 3 条硬政策"
- "本单位晋升，学历在这 4 个场合起作用"
- "教师/医护读研，双证才是职称评审认可的"

### 维度 3：按学历

**分类选项**（对应 Q4）：

| 分类 | 对应策略 |
|------|---------|
| 全日制本科 | 巩固型：让优势更明显 |
| 非全日制本科 | 补短板型：不让学历卡住 |
| 大专 | 先补本科，才能进入研究生 |

**每个学历起点的内容侧重：**

- 这个起点在体制内**常见的困境**
- 有哪些**可选路径和不可选路径**
- 常见**陷阱**（比如大专直接考党校的问题、非全本科被歧视的坑）
- 这类人**真实选择的分布**
- **推荐的行动路径**

**示例：非全日制本科**
- 政策依据：自考、成教、函授、网教、国开均属国民教育序列
- 实际作用：考公务员、考研、考事业编均承认；但单位内"隐性歧视"存在
- 学员数据：研知道 1,123 人中，48% 是非全日制本科学员
- 推荐路径：四川党校 / 重庆党校 / 部分 MPA 院校（无本科年限要求的）

### 维度 4：按年龄

**分类选项**（对应 Q5）：

| 分类 | 关键政策窗口 |
|------|-----------|
| 25-30 岁 | 5 年服务期 / 入职初期 |
| 31-35 岁 | 重庆遴选上限 / 职级晋升关键期 |
| 36-40 岁 | 四川遴选上限 / 最后机会窗口 |
| 41 岁+ | 延退觉醒 / 职称评审最后机会 |

**每个年龄段的内容侧重：**

- 这个年龄段的**政策窗口**（能做什么、不能做什么的硬线）
- 读研的**时间成本和机会成本**
- 这个年龄段的**典型案例**
- 和同龄人的**对比数据**

**示例：36-40 岁**
- 政策窗口：四川遴选上限 40 岁（2025 年公告）
- 时间成本：党校 3 年 → 读完 39-43 岁，错过重庆遴选但抓住四川
- 案例：李姐 38 岁医院后勤岗，选省委党校政治学
- 数据：研知道该年龄段学员中，65% 以"防御性学历"为目标

## 四、浏览详情页的统一结构

每个浏览详情页 = 5 段固定结构 + 底部 CTA

```
┌──────────────────────────────────────┐
│  ← 返回   [维度 · 具体选项]           │
│                                      │
│  ── 政策依据 ──                      │
│  （3-5 条政策文件 + 条款 + 解释）       │
│                                      │
│  ── 读研在这个 [x] 的实际作用 ──      │
│  （分晋升 / 涨工资 / 遴选三类）        │
│                                      │
│  ── 和你一样的人怎么选的 ──           │
│  （统计分布 + 进度条 + 关键数字）       │
│                                      │
│  ── 典型学员 ──                      │
│  （1-2 个代表性案例 + 展开更多）        │
│                                      │
│  ── 这个 [x] 的推荐路径 ──            │
│  （首选 + 备选 + 一句话理由）           │
│                                      │
│  ┌────────────────────────────────┐  │
│  │  测一测我适合哪条                │  │
│  │  → 6 题 3 分钟出方案            │  │
│  └────────────────────────────────┘  │
└──────────────────────────────────────┘
```

### 每段的字数控制

| 段 | 字数 | 阅读时间 |
|----|------|---------|
| 政策依据 | 150-250 字 | 30 秒 |
| 实际作用 | 200-300 字 | 45 秒 |
| 同行数据 | 100 字 + 图示 | 20 秒 |
| 典型学员 | 150 字（单个案例） | 30 秒 |
| 推荐路径 | 80 字 | 15 秒 |
| **总计** | ~800 字 | **2-3 分钟** |

单页阅读时间控制在 3 分钟以内，比答题用时略长但不超过太多。

## 六、完整内容示例：财政系统

**作为所有系统类详情页的写作模板。**

```markdown
## 政策依据

1. **《公务员职务与职级并行规定》**（2019 中办发文）
   明确研究生学历在职级并行起始档位中的加分作用。财政系统干部的职级晋升走职务和职级两条线，研究生学历在后者有直接影响。

2. **四川省省直机关遴选公务员公告（2025 年度）**
   省财政厅、省税务局等财税口省直单位，50% 以上岗位要求研究生学历。

3. **《组织考察办法（县处级）》**
   学历作为"综合素质"考察的硬指标之一，在同等条件下具有决定性作用。

## 读研在财政系统的实际作用

**晋升**
- 竞争上岗：研究生学历 +0.5 档（同等条件下优先）
- 职级并行：研究生起步档位高于本科
- 组织考察：学历作为综合素质的硬指标

**涨工资**
- 研究生基本工资起步加薪（约 500-800 元/月，视地区）
- 绩效考核中学历权重提升（部分单位明确量化）

**遴选**
- 省直财税遴选：双证是硬门槛（党校学历不认）
- 市直财税遴选：研究生学历优先（非硬门槛但实际中权重高）

## 和你一样的人怎么选的

基于 186 位财政系统学员数据：

- 78% 选了党校（省委党校经济学为主）
- 22% 选了 MPA（西南财经大学为主）
- 平均年龄：33 岁
- 平均工龄：8 年
- 最常见目标：本单位晋升（62%）
```

## 七、路径对比页内容结构

```markdown
## 党校 vs MPA vs MBA — 核心对比表

| 维度 | 党校 | MPA | MBA |
|------|------|-----|-----|
| 证书 | 单证（学位） | 双证（学位+学历） | 双证 |
| 学费 | 2-3 万 | 5-10 万 | 10-30 万 |
| 学制 | 3 年 | 2-3 年 | 2-3 年 |
| 考试 | 政治+专业+综合 | 管综+英语 | 管综+英语 |
| 难度 | 中（不考英数） | 较高（考英数） | 较高 |
| 遴选认可 | ❌ | ✅ | ✅ |
| 调任认可 | ❌ | ✅ | ✅ |
| 职称评审 | ❌ | ✅ | ✅ |
| 本单位晋升 | ✅ | ✅ | ✅ |

## 哪些场景谁更合适

- **本单位晋升** → 党校 > MPA（性价比）
- **遴选转公** → MPA > 党校（硬门槛）
- **职称评审**（教师/医护/专技岗） → MPA > 党校（党校不认）
- **国企管理岗** → MBA > MPA（企业系统认可度高）
- **怕英数统考** → 党校
- **预算 < 3 万** → 党校
- **想快速拿证**（2 年制） → MPA

## 政策认可度详对比

| 场景 | 党校 | MPA/MBA | 依据 |
|------|------|--------|------|
| 职级并行 | ✅ | ✅ | 2019 中办发文 |
| 竞争上岗 | ✅ | ✅ | 职务与职级并行规定 |
| 省直遴选 | ❌ | ✅ | 2025 四川遴选公告 |
| 市级遴选 | ❌ | ✅ | 2025 重庆遴选公告 |
| 调任 | ❌ | ✅ | 公务员调任规定 |
| 职称评审 | ❌ | ✅ | 人社部职称制度改革意见 |
| 组织考察 | ✅ | ✅ | 县处级组织考察办法 |
| 干部培训 | ✅ | ✅ | 党校条例 |

## 常见误区

- **误区 1：党校学历不如 MPA**
  错。本单位晋升、职级并行、组织考察都认。只是特定场合（遴选/调任/职称）不认。

- **误区 2：MBA 比 MPA"高级"**
  取决于你的系统。企业管理岗 MBA 认可度高，体制内行政岗 MPA 更对口。

- **误区 3：党校不是"真研究生"**
  党校研究生学历在中发〔2000〕10 号文明确认可，教育部 1995 年复函确认。在职级并行、职称评审之外的领域均有作用。
```

## 八、案例库页内容结构

### 列表页

每条卡片显示：
- 姓名（化名）· 年龄 · 地区 · 单位
- 学历 · 工龄 · 党员身份
- 选了什么（学校+专业）
- 一句话引语
- 标签（系统 / 学历 / 类型）

### 筛选维度

- 按系统（复用维度 1 分类）
- 按学历（复用维度 3 分类）
- 按年龄（复用维度 4 分类）
- 按选择（党校 / MPA / MBA）

### 案例详情页

点击卡片进入详情，展开完整内容：
- 基本画像
- 决策动机（为什么读研）
- 选择过程（纠结点 / 最终决定的原因）
- 选择结果（在读 / 已上岸 / 备考中）
- 结构化要点（学费 / 学制 / 通勤 / 备考投入）

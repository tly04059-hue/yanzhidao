## 一、诊断目标

通过 8 道题收集用户画像，输出：
1. 首选路径（学校 + 专业 + 理由 + 匹配度）
2. 备选路径
3. 最匹配的真人案例
4. 关键提醒信息（报名时间/考试形式/过考率）
5. 触发的警告/提示

---

## 二、8 道题的数据流

| 题 | key | 选项 | 影响范围 |
|----|-----|------|---------|
| Q1 | system | 党政机关/公检法/乡镇街道/教育医疗/国企银行/民族地区 | Q6 选项、Q7 选项、专业匹配、院校推荐 |
| Q2 | party_member | 是/否 | 非党员+四川 → 排除四川党校 |
| Q3 | region | 四川/重庆 | 院校范围、党校选择 |
| Q4 | education | 全日制本科/非全日制本科/大专 | 大专阻断；话术切换 |
| Q5 | age | 25-30/31-35/36-40/41+ | 遴选年龄提醒、案例匹配 |
| Q6 | goal | 动态（核心分流） | **MPA vs 党校** |
| Q7 | position | 动态（按系统） | 专业方向精确匹配 |
| Q8 | budget | 2-3万/5-8万/8万+ | 预算兜底交换 |

## 三、三层决策树

### 第一层：目标分流（Q6 → 单证 or 双证）

| 目标 | 首选方向 | 原因 |
|------|---------|------|
| 遴选 | MPA 优先 | 遴选要求国民教育序列，党校不认 |
| 转公务员 | MPA 优先 | 调任需双证（《公务员调任规定》中组发〔2019〕21号） |
| 职称晋升 | MPA 优先 | 职称评审不认党校学历 |
| 本单位晋升 | 党校优先 | 竞争上岗/职级并行认可，性价比最高 |
| 防御性学历 | 党校优先 | 同上 |

### 第二层：资格过滤（9 条排除/提示规则）

> **工程权威**：详见 `06-推荐逻辑规则表.md §五 21 规则单元`（v0.6 · B/E/P/R/A/DP 6 类型 · 含每条 trigger / output / priority / dependency 字段）。本表是业务叙述视角的简化版，工程实施请以 06 为准。

| # | 条件 | 动作 | 政策依据 |
|---|------|------|---------|
| 1 | 非党员 + 四川 | 排除四川党校 → 推重庆党校（需在川工作满2年+本科） | 四川党校招生简章 |
| 2 | 大专学历 | 硬阻断 → 跳结果页引导先升本科 | 四川党校不接受单大专 |
| 3 | 目标=遴选 | 提示：遴选不认党校，必须双证 MPA | — |
| 4 | 教育医疗 + 教师/医护 | 提示：党校学历在职称评审中不认，双证可缩短2-3年 | 人社部《深化职称制度改革的意见》 |
| 5a | 重庆 + 遴选 + 36岁+ | 提示：重庆遴选上限 35 岁 | 2025年重庆遴选公告 |
| 5b | 四川 + 遴选 + 41岁+ | 提示：四川遴选上限 40 岁 | 2025年四川遴选公告 |
| 6 | 目标=转公务员 | 提示：调任需副处/副高满2年+MPA 是基础件 | 《公务员调任规定》 |
| 7 | 推荐法学时 | 提示：法学招生仅 100 人，过考率 6.9% 最低 | 四川党校统计 |
| 8 | 乡镇街道 + 25-30岁 | 提示：5年服务期内不能调动/遴选，读研是唯一能做的晋升准备 | — |
| 9 | 事业编 + 专技岗 | 提示：职称不认党校学历，强制 MPA 优先 | — |

**附加规则：**
- MPA 首选时追加提示："川渝范围推荐，全国范围可1对1咨询"
- 预算 2-3万 + MPA 首选 → 交换（党校变首选，MPA 变备选）

### 第三层：具体匹配

根据系统 + 岗位 → 推荐具体学校和专业。

> **运行时权威**：当前推荐引擎已切换到 `data/derived-from-context/strategy_pool.json` 的 14 条策略（含 `school_query` / `school_priorities` / `risk_cards` 字段）。本表是 v2.0 业务叙述视角的"系统→专业"映射，作长版参考保留，运行时不再消费。

### 四川党校（3 专业 12 方向）

| 岗位/系统 | 推荐专业 | 方向 |
|----------|---------|------|
| 组织部/宣传部/统战部/机关党委 | 政治学 | 党建 |
| 公安系统 | 政治学 | 社会治理 |
| 法院/检察院 | 法学 | 法治社会 |
| 纪委监委 | 政治学 | 社会治理（备选：法学·党内法规） |
| 应急管理 | 政治学 | 应急管理 |
| 民政/人社/街道/社区 | 政治学 | 公共管理 |
| 财政/经信/发改/统计 | 经济学 | 区域经济 |
| 法治/信访/司法 | 法学 | 法治社会 |
| 纪检+党内法规 | 法学 | 党内法规 |
| 无明确对口 | 政治学 | 公共管理（默认） |

**专业选择提示：**
- 法学招生仅 100 人，过考率 6.9%（最低）→ 无基础不推荐
- 经济学不是西方经济学，不考数学 → 自动说明
- 政治学招 300 人，过考率 10.9%（最高）→ 作为"安全牌"

### 重庆党校（5 专业）

| 岗位/系统         | 推荐专业     |
| ------------- | -------- |
| 组织/宣传/纪委/国企党委 | 党政管理     |
| 公安系统          | 公共管理     |
| 法院/检察院        | 法律       |
| 民政/人社/应急/街道   | 公共管理     |
| 发改/政研/大型国企战略  | 战略管理     |
| 财政/经济/金融      | 经济管理     |
| 无明确对口         | 公共管理（默认） |

**提示：**
- 公管和党政参考书最薄 → 备考资料量最小
- 各专业过考率 12%-14.7%，差异不大
- 党政=管人，公管=管事

### MPA/MBA 院校匹配

| 系统+地区 | 推荐院校 | 逻辑 |
|----------|---------|------|
| 四川省厅/成都市直 | 四川大学 MPA | 省市两级校友最强 |
| 财税/金融系统 | 西南财经大学 MPA | 系统校友统治力 |
| 交通/住建/水利 | 西南交通大学 MPA | 行业对口+2年制 |
| 重庆市直/区县 | 重庆大学 MPA | 重庆系统最高认可 |
| 教育系统 | 四川师范/西南大学 | 系统内校友密度 |
| 国企管理岗 | MBA（川大/电子科大/重大） | 企业系统 MBA 认可度高于 MPA |

## 五、推荐理由话术系统

> **运行时权威**：当前推荐 reason 文案已迁移到 `strategy_pool.json` 14 条策略的 `reason` 字段（按 systems / goals / age_bands / budget_max 匹配触发对应 reason）。本节"全日制本科 / 非全日制本科"两挡话术是 v2.0 业务版，作历史参考。

根据用户学历类型自动切换话术风格：

| 学历 | 党校推荐话术 | MPA 推荐话术 |
|------|------------|------------|
| 全日制本科 | "让你的优势更明显。巩固现有条件，在组织考察和竞争上岗中更有底气。" | "双证MPA让你的学历优势扩展到更多场景。" |
| 非全日制本科 | "不让学历成为你唯一的短板。档案里多一笔实打实的研究生学历，在职级并行和竞争上岗中不被比下去。" | "双证硕士是从非全日制本科到研究生学历的质变。" |

来源：人群洞察报告 — 52% 全日制为"巩固型"，48% 非全日制为"补短板型"。

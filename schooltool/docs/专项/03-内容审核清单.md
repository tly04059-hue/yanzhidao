## 二、问题与选项设计

### Q1 省份
- 四川 / 重庆
- 作用：决定默认院校范围，影响党校选择

### Q2 体制类型
- 公务员（行政编制）
- 参公事业编（参照公务员管理）
- 事业编（全额/差额拨款）
- 教育/医疗（教师、医护人员）
- 国企/银行（国有企业、城投、银行）
- 乡镇/街道（基层工作人员）
- 作用：影响岗位选项（Q8）、专业匹配、院校推荐

### Q3 最高学历
- 全日制本科（统招本科）
- 非全日制本科（自考/成考/网教/电大）
- 大专（全日制或非全日制大专）→ **硬性阻断，直接跳结果**
- 作用：大专直接阻断；非全日制本科影响案例匹配

### Q4 党员身份
- 是（含预备党员）
- 不是
- 作用：非党员排除四川党校，推荐重庆党校（需满足在川工作满2年+本科学历）

### Q5 年龄段
- 25-30岁 / 31-35岁 / 36-40岁 / 41岁以上
- 作用：影响政策提示（延退）、遴选年龄限制提醒、案例匹配

### Q6 核心目标
- 本单位晋升（竞争上岗、职级并行）
- 遴选到上级机关（省级/市级遴选）
- 职称评审（评副高/正高需要硕士）
- 想转公务员（国企/事业编调任行政编）
- 防御性学历（周围人都在考，不能落后）
- 作用：核心分流逻辑，决定MPA优先还是党校优先

### Q7 学费预算
- 2-3万（党校在职研究生）
- 5-8万（MPA/教育硕士）
- 8万以上（MBA/名校MPA）
- 作用：预算2-3万+MPA首选时，交换首选备选

### Q8 岗位方向（动态选项，依赖Q2）

**公务员/参公：**
组织/干部管理、宣传/文化、纪委监委、**公安系统**、**法院/检察院**、财政/审计、发改/经济、司法/法治/信访、应急管理、人社/民政、综合管理岗

**教育/医疗：**
教师、医护人员、学校/医院党务岗、综合管理岗

**国企/银行：**
管理岗、技术岗、金融/业务岗、党建/纪检、综合管理岗

**乡镇/街道：**
社区/村镇管理、社会事务/民政、党建/组织、综合管理岗

**事业编：**
**管理岗**（走职务晋升路线）、**专技岗**（走职称评审路线）、党务/行政、财务/审计、综合管理岗

## 三、推荐逻辑规则

### 3.1 排除/警告规则（共7条）

| # | 触发条件 | 类型 | 提示内容 |
|---|---------|------|---------|
| 1 | 四川+非党员 | 阻断 | 四川党校要求党员，推荐重庆党校（需在川工作满2年+本科学历） |
| 2 | 目标=遴选 | 信息 | 遴选只认国民教育序列（双证），党校不认 |
| 3 | 教育医疗系统+教师/医护 | 信息 | 党校学历在职称评审中不认可，双证可缩短2-3年 |
| 4 | 大专学历 | 硬阻断 | 直接跳结果页，引导先提升本科 |
| 5 | **重庆+遴选+36岁以上** | 信息 | 重庆市级遴选上限35岁（2025年公告），建议转向本单位晋升 |
| 5b | **四川+遴选+41岁以上** | 信息 | 四川省直遴选上限40岁，中央遴选科员38/副科43/处级46 |
| 6 | 目标=转公务员 | 信息 | 调任需副处级或副高+组织推荐，MPA双证是基础件 |
| 7 | 法学推荐时 | 注意 | 法学招生仅100人，过考率6.9%最低 |
| **8** | **乡镇街道+25-30岁** | 信息 | 5年服务期内不能调动/遴选，读研是服务期内唯一能做的晋升准备 |
| **9** | **事业编+专技岗** | 信息 | 走职称路线，党校学历在职称评审中不认，强制MPA优先 |

**新增逻辑：推荐理由按学历类型切换话术**
- 全日制本科 → "让你的优势更明显"（巩固型话术）
- 非全日制本科 → "不让学历成为你唯一的短板"（补短板型话术）

### 3.2 首选/备选推荐逻辑

```
目标 = 遴选/转公务员 → MPA优先
  └ 备选：党校（如符合条件）

目标 = 职称晋升 → MPA优先（职称不认党校）
  └ 备选：党校（走管理岗的话）

目标 = 本单位晋升/防御 →
  ├ 四川+党员 → 四川党校优先，备选MPA
  ├ 重庆 → 重庆党校优先，备选MPA
  └ 四川+非党员 → 重庆党校优先，备选MPA

预算覆盖：2-3万 + MPA首选 → 交换（党校变首选）
```

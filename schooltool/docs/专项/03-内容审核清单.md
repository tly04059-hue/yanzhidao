## 零、内容生产纪律 6 组硬条款（v0.2 新增 · 必须先通过才能上线）

> 所有内容上线前必须通过本节 6 组硬条款的 AI 预审 + 人工抽查（PRD v3.1 §九 内容验收的"AI 预审 + 人工抽查"机制）。命中违规 → 阻止上线，回炉重写。

### 0.1 · ER1 不打"主力客群"标签

- **约束**：文案不得含"主力客群 / 最强 / 最集中 / 谁占多数 / 主要面向"等暗示概率分布的措辞
- **判断依据**：主力客群描述以客户管理客户资料的清单分布为依据，不靠抽检/AI 推导/惯性判断
- **允许的措辞**："遇到该场景的客户全部适用" / "对应场景"
- **依据**：经营战略总纲 §5.2 公理第 ⑥ 类 / M4 §八 / M7 §七 R10 / 06 v0.3 §四 ER1

### 0.2 · ER2 政策口径用"可能性和判断方向"

- **约束**：所有政策结论用"基于核实政策事实的可能性和判断方向"措辞，**不做确定的因果论断**
- **可用措辞**："政策支持的可能性" / "判断方向" / "可能的应用场景" / "对应的政策依据"
- **禁用措辞**："一定会" / "肯定能" / "必然导致" / "保证升职/调任/落编" / "100%"
- **依据**：上下文价值观层 §3.8 C2 / M7 §七 R11 / 06 v0.3 §四 ER2

### 0.3 · ER3 三个硬约束（v3.1 校准）

- **不催单**：禁用"今天最后一天 / 名额仅剩 / 立即报名 / 倒计时 / 马上就要截止"
- **不强制公开传播**：禁用"分享朋友圈解锁 / 转发到群解锁 / 分享换权益 / 分享得礼包"——用户主动发到朋友圈是 OK 的，工具不阻止；禁止的是机构强制/利益换分享
- **不让朋友圈尴尬**：私域承接为主（保存到手机 / 私聊朋友/信任的人），公域裂变慎用；用户自决
- **大大方方说分享**（v3.1 校准）：分享/海报功能保留，按钮明示"分享给自己 / 分享给朋友 / 分享给信任的人 / 分享海报"——不藏不诱
- **依据**：M5 §五.2 / 业务必读 §5.1 / 内容生产手册 §一.3 / M7 §七 R12 / 06 v0.3 §四 ER3 / ww 2026-05-18 PRD v3.1 评审校准

### 0.4 · 价值观层 §5.4 禁用词清单

| 类别 | 禁用词 | 替代措辞 |
|---|---|---|
| 空洞承诺 | 助力 / 赋能 / 保驾护航 / 一站式 / 全方位 | 说具体做什么 |
| 绝对化 | 保过 / 必过 / 100% / 最好 / 唯一 | 用具体数据 |
| 机构口吻 | 我们致力于 / 我们的使命 / 我们的愿景 | ww 第一人称 |
| 品牌自嗨 | 研知道学员 / 研知道家人 / 学姐学长 | 直呼名字或"你" |
| 焦虑推手 | 再不 X 就来不及了 / 别人都已经… | 客观描述窗口期 |
| 推卸判断 | 看个人情况 / 仁者见仁 / 各有利弊 | 给明确判断+理由 |

### 0.5 · 价值观层 §5.3 反面禁止（声音特征级）

- 机构官方口吻（"研知道致力于为您提供专业的考研辅导服务"）
- 空洞承诺（"助力您实现梦想 / 陪伴您的考研之旅"）
- 焦虑营销（"再不考就来不及了 / AI 要抢走你的饭碗"）

- 和稀泥（"各有利弊 / 看个人情况 / 仁者见仁"）
- 品牌视角（"研知道学员喜报 / 让我看看研知道的同学们"）

### 0.6 · 防御性表述纪律（v3.1 新增）

- **必须用**：保留可能性 / 不掉队 / 占位 / 防御性资产 / 准入项过渡 / 确定性投资

## 二、问题与选项设计

### Q1 省份
- 四川 / 重庆
- 作用：决定默认院校范围，影响党校选择

### Q2 体制类型
- 公务员（行政编制）
- 参公事业编（参照公务员管理）
- 事业编（全额/差额拨款）
- 教育/医疗（教师、医护人员）
- 国企/银行（国有企业、城投、银行）
- 乡镇/街道（基层工作人员）
- 作用：影响岗位选项（Q8）、专业匹配、院校推荐

### Q3 最高学历
- 全日制本科（统招本科）
- 非全日制本科（自考/成考/网教/电大）
- 大专（全日制或非全日制大专）→ **硬性阻断，直接跳结果**
- 作用：大专直接阻断；非全日制本科影响案例匹配

### Q4 党员身份
- 是（含预备党员）
- 不是
- 作用：非党员排除四川党校，推荐重庆党校（需满足在川工作满2年+本科学历）

### Q5 年龄段

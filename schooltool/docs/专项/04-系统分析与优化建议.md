> 基于体制内人事制度研究 + 川渝遴选政策专项研究，对工具的分类、逻辑、信息进行全面审视。

## 一、Q2体制类型分类评估

### 当前分类 vs 实际编制体系

| 当前Q2选项 | 对应的实际编制     | 问题                       |
| ------ | ----------- | ------------------------ |
| 公务员    | 行政编制（综合管理类） | OK，但缺少行政执法类              |
| 参公事业编  | 事业编制（参公管理）  | OK                       |
| 事业编    | 事业编制-管理岗    | OK，但未区分管理岗/专技岗           |
| 教育/医疗  | 事业编制-专技岗    | 本质是事业编的子集，但晋升路径完全不同，单列合理 |
| 国企/银行  | 企业编制        | OK                       |
| 乡镇/街道  | 不是编制类型，是层级  | **有问题**                  |

### 核心问题

**1. 乡镇/街道是层级不是系统**

乡镇公务员还是公务员，乡镇事业编还是事业编。把它和公务员并列是分类错误。

但从业务角度看，乡镇/街道人员有几个共同特征：
- 服务期限制（新录用乡镇公务员5年不能遴选）
- 基层经历要求（遴选加分项）
- 预算敏感（收入普遍偏低）
- 晋升天花板低（多数止步副科）

**建议**：保留乡镇/街道选项，但改为问法："你目前在哪一级单位工作？"或在选公务员/事业编后，追问是否在乡镇街道。当前的做法虽然分类学上不严格，但对于快速筛选用户画像是有效的。**保留，但在文案上调整**。

**2. 缺少公检法司**

公检法司（政法专项编制）有独特的晋升路径：
- 法官/检察官：走员额制，有单独的等级序列
- 公安：警衔 + 行政职级双轨
- 区别于普通公务员，读研的目的也不同

之前的对照表HTML里有"纪检/公检法司"的单独分类。

**建议**：在公务员/参公的岗位选项（Q8）中细分，而不是在Q2增加选项。因为公检法司的编制本质还是公务员（行政编制），区别在岗位。

**3. 事业编的管理岗/专技岗区分不够**

事业编管理岗走行政级别，专技岗走职称。两条路径对学历的需求完全不同：
- 管理岗：党校学历认可，MPA加分
- 专技岗：必须双证，职称评审不认党校

**建议**：在Q6（目标）或Q8（岗位）中区分，不在Q2层面增加复杂度。

### 最终判断

**Q2的6个选项总体合理，不建议大改。** 需要做的优化：
1. 乡镇/街道的文案改为"乡镇/街道（基层）"，副标题改为"不论编制类型，目前在乡镇或街道工作" — 待前端文案调整
2. ✅ Q8公务员的岗位选项增加"公安系统"和"法院/检察院" — 已完成
3. ✅ Q8事业编的岗位选项区分"管理岗"和"专技岗" — 已完成

## 二、推荐理由的逻辑深度不够

### 当前问题

推荐理由太简单，像"着眼党的队伍和组织体系建设，干部培训对象优先"——这只是描述专业内容，没有回答"为什么这个选择对你最好"。

### 推荐理由应覆盖的维度

每条推荐理由应该回答以下问题（不是全部写出来，而是根据用户画像选择最相关的2-3点）：

| 维度 | 说明 | 示例 |
|------|------|------|
| **职业目标匹配** | 这个学历如何服务于你的核心目标 | "遴选只认国民教育序列学历，MPA双证是报名硬门槛" |
| **制度依据** | 引用具体政策或制度 | "根据《公务员调任规定》，事业编调任公务员需副高满2年或正处以上" |
| **竞争优势** | 学历带来的具体优势是什么 | "硕士定级四级主任科员，比本科高2个职级，相当于提前4年" |
| **时间价值** | 什么时候读最划算 | "31-35岁是最佳窗口：遴选窗口还开着，延退后还有25+年回报期" |
| **预算效率** | 为什么这个价位最合理 | "党校2.16万 vs MPA 8.4万。你不遴选，省下的6万没有额外回报" |
| **风险提示** | 选错的后果 | "党校学历在遴选中不被认可，如果3年后想遴选会后悔" |
| **政策趋势** | 当前政策环境如何影响决策 | "延迟退休已在推进，37岁读研还有23年用；两会'投资于人'信号明确" |
| **案例佐证** | 类似的人怎么选的 | "跟你情况差不多的张姐选了政治学（党建），组织考察时学历是过关项" |

### 当前engine.js推荐理由的改进方向

**四川党校推荐时**，理由应包含：
- 为什么选这个专业方向（岗位对口性）
- 党校学历在你的目标场景下的认可度（晋升认/遴选不认）
- 考试难度提示（排名制，政治学录取率最高10.9%）
- 时间和预算的性价比

**MPA推荐时**，理由应包含：
- 双证的制度价值（遴选、调任、职称评审全认可）
- 为什么推荐这个具体院校（校友网络/系统对口）
- 川渝范围默认推荐，全国范围可在1对1中提供

**重庆党校推荐给四川人时**，理由应包含：
- 报考条件完整描述（在川工作满2年+本科学历）
- 与四川党校的对比（不要求党员、学费差2400、专业设置差异）
- 跨省报考的政策依据

| 触发条件 | 状态 | 提示内容 |
|---------|------|---------|
| 乡镇/街道 + 25-30岁 | ✅ | 5年服务期内不能调动/遴选，读研是唯一能做的晋升准备 |

| 事业编 + 专技岗 | ✅ | 职称不认党校，强制MPA优先 |
| 待做：专技岗理由量化 | 📋 | "硕士比本科评副高缩短3-5年，按副高与中级的工资差（约1500-2000元/月），3年差额约5-7万，远超学费投入。" |

### 5.2 推荐理由模板升级

每条推荐理由应按以下结构生成：

```
[职业目标匹配] + [制度依据/数据支撑] + [时间或预算价值]
```

示例（四川公务员+党员+本单位晋升+组织部）：
> 组织部工作与党建方向直接对口。党校学历在竞争上岗、职级并行、干部考察中均被认可（中发〔2000〕10号）。政治学招生300人，过考率10.9%为三个专业中最高，研知道入班辅导过考率40.4%，1对1辅导过考率86.3%。学费2.16万，性价比远高于MPA。

示例（四川公务员+遴选+31-35岁+财政局）：
> 遴选要求国民教育序列学历（学信网可查），党校在职研究生不满足条件。西南财经大学在财税金融系统的校友网络是川渝最强的，遴选面试环节校友背书有隐性价值。四川遴选年龄上限40岁，你31-35岁读完MPA（2-3年）后仍有充足的遴选窗口。

### 5.3 政策引用应具体化

当前的政策引用太笼统（"两会信号"）。应引用具体文件：

| 场景 | 应引用 |
|------|--------|
| 职级并行 | 2019年中办《公务员职务与职级并行规定》 |
| 党校学历认可 | 中发〔2000〕10号 + 教育部1995年复函 |
| 调任条件 | 《公务员调任规定》（中组发〔2019〕21号） |
| 遴选条件 | 当年四川/重庆遴选公告原文 |
| 职称评审 | 人社部《深化职称制度改革的意见》+ 四川省实施方案 |
| 延迟退休 | 2025年渐进式延迟法定退休年龄办法 |

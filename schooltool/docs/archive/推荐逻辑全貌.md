## 三层决策树

### 第一层：目标分流（Q6 → 双证 or 单证）

| 目标 | 首选方向 | 原因 |
|------|---------|------|
| 遴选 | MPA优先 | 遴选要求国民教育序列学历，党校不认 |
| 转公务员 | MPA优先 | 调任需双证（《公务员调任规定》） |
| 职称晋升 | MPA优先 | 职称评审不认党校学历 |
| 本单位晋升 | 党校优先 | 竞争上岗/职级并行认可，性价比最高 |
| 防御性学历 | 党校优先 | 同上 |

### 第二层：资格过滤（排除不符合的选项）

| 条件 | 动作 |
|------|------|
| 非党员+四川 | 排除四川党校→推重庆党校（需在川工作满2年+本科学历） |
| 大专学历 | 硬阻断→直接跳结果页引导升本科 |
| 教师/医护+党校 | 提醒职称不认党校 |
| 重庆+遴选+36岁以上 | 提醒重庆遴选上限35岁 |
| 四川+遴选+41岁以上 | 提醒四川遴选上限40岁 |
| 国企+转公务员 | 提醒调任需副处或副高满2年+MPA是基础件 |
| 法学推荐时 | 提醒招生仅100人，过考率6.9%最低 |
| 乡镇+25-30岁 | 提醒5年服务期，读研是服务期内唯一晋升准备 |
| 事业编+专技岗 | 提醒职称不认党校，强制MPA优先 |
| MPA首选时 | 追加提示"川渝范围推荐，全国范围可1对1咨询" |

### 第三层：具体匹配

**党校路径：**
- 四川党校：岗位→3个专业（政治学/经济学/法学）→12个方向
- 重庆党校：岗位→5个专业

**MPA路径：**
- 财税金融→西南财经
- 交通住建→西南交大
- 教育系统→四川师范（教育硕士）
- 国企管理→川大MBA
- 重庆→重庆大学
- 默认→四川大学

**预算兜底：** 预算2-3万 + MPA首选 → 交换（党校变首选）

## 8道题的数据流

| 题 | key | 影响 |
|----|-----|------|
| Q1 省份 | region | 决定院校范围，影响党校选择 |
| Q2 体制 | system | 影响Q8选项、专业匹配、院校推荐 |
| Q3 学历 | education | 大专阻断；非全日制影响案例匹配 |
| Q4 党员 | party_member | 非党员排除四川党校 |
| Q5 年龄 | age | 遴选年龄提醒、政策匹配、案例匹配 |
| Q6 目标 | goal | **核心分流**：MPA vs 党校 |
| Q7 预算 | budget | 预算兜底交换 |
| Q8 岗位 | position | 专业方向匹配、MPA院校匹配 |

## 案例匹配权重

体制类型30% + 目标25% + 学历20% + 地区15% + 年龄10% + 非党员加20

## 已修复的逻辑缺陷（2026-03-10）

1. ✅ 遴选年龄已区分川渝（四川40→41+提醒，重庆35→36+提醒）
2. ✅ 乡镇+25-30岁触发5年服务期提醒
3. ✅ 事业编Q8新增管理岗/专技岗选项，专技岗强制MPA优先+提醒职称不认党校
4. ✅ Q8新增公安系统、法院/检察院选项+专业匹配规则
5. ✅ 推荐理由按学历类型切换话术（全日制→巩固型，非全日制→补短板型）

## 关键政策依据

| 场景 | 依据文件 |
|------|---------|
| 职级并行 | 2019年中办《公务员职务与职级并行规定》 |
| 党校学历认可 | 中发〔2000〕10号 + 教育部1995年复函 |
| 调任条件 | 《公务员调任规定》中组发〔2019〕21号 |
| 四川遴选 | 2025年度四川省省直机关遴选公告（40岁/本科以上） |
| 重庆遴选 | 2025年度重庆市遴选公告（35岁/302人） |
| 职称评审 | 人社部《深化职称制度改革的意见》 |

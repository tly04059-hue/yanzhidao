# Archive 说明

`docs/archive/` 只存放历史版本、早期方案和已退出主线的文档，不作为当前协作主文档。

## 归档类型

- `已吸收归档`
  旧文档的核心结构和主要结论，已经整理进当前主文档。后续讨论和更新以当前主文档为准，旧稿仅保留备查。
- `参考归档`
  旧文档已退出主线，但仍保留补充背景、推导过程、早期方案细节或表达素材。当前主文档未必完整覆盖其全部内容，需要时仍可回看。

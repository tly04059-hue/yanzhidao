# 政策表述标准（PD 三档说法）

## 一、用途与范围

**用途**：所有政策表述以本表为唯一源头。改动政策事实时**先改本表**，再批量对账下游文档。

**范围**：

- 选校工具 docs / data / 原型 HTML（含 4 个当前原型 + 旧版原型）
- L1 跨模块的政策引用（M1-M9 + 价值观层）
- 对外内容：小程序、推文、公众号、销售脚本、宣讲口播稿、私域内容

**不在范围**：

- 战略层文档（商业全景分析、上下文价值观层"正例"段）按业务自由表述
- 归档类（_archived / claim-candidates.jsonl / skills 历史 references）保留原状作时间快照

## 二、三档说法规则

| 档位 | 长度 | 适用场景 |
|---|---|---|
| **短**（≤20 字）| 一句话 | 卡片副标题 / 政策卡 title / 列表项 / ASCII 图占位 / 短视频文案小标题 |
| **中**（≤80 字）| 一段话 | 卡片正文 / one_liner / 政策表行 / 推文小标题 / 销售一句话回应 |
| **长**（≤200 字）| 完整段落 | 政策详情页 / detail 字段 / 政策原文引用段 / 推文正文段 / 宣讲口播展开 |

## 三、PD 三档说法表（11 条）

### PD-01 · 公务员职务与职级并行规定 + 新录用任职定级规定

> **重要**：研究生学历的两层政策依据分别来自两份不同文件，本 PD 合并表述。

- **文号**：中办发〔2019〕21 号《公务员职务与职级并行规定》+ 中组发〔2019〕10 号《新录用公务员任职定级规定》
- **短**：职级并行学历优势
- **中**：2019 中办《职务与职级并行规定》+ 中组发《新录用任职定级规定》——研究生学历在新录用起点定级享有起点优势，在职级晋升综合评价中是重要维度
- **长**：2019 年中央办公厅印发《公务员职务与职级并行规定》（中办发〔2019〕21 号）+ 中央组织部《新录用公务员任职定级规定》（中组发〔2019〕10 号），研究生学历在新录用任职定级中享有起点优势，在职级晋升综合评价中是重要维度；近年部分系统遴选/调任岗位出现"研究生为基本门槛"的趋势（基于公开公告的判断方向，具体晋升视个人单位情况）。

### PD-02 · 党校学历认可文件

- **文号**：中发〔2000〕10 号《中共中央关于面向 21 世纪加强和改进党校工作的决定》第（二十八）条
- **短**：党校学历体制内认可
- **中**：中发〔2000〕10 号文规定党校学历"待遇参照国民教育相应学历的有关规定"，体制内党政机关 / 事业单位 / 国有企业内部认可
- **长**：中发〔2000〕10 号《中共中央关于面向 21 世纪加强和改进党校工作的决定》第（二十八）条明确"取得党校学历，待遇参照国民教育相应学历的有关规定"。学信网不可查 ≠ 不被承认，在体制内本单位晋升、内部竞争、职务任免场景有效；跨系统使用（遴选 / 调任 / 部分专业职称评审）存在限制（具体执行视调入机关和岗位规定）。

### PD-03 · 公务员调任规定

- **文号**：中组发〔2019〕21 号《公务员调任规定》（中央组织部 2008 年制定，2019-10-15 修订，2019-11-26 施行）
- **短**：调任需国民教育学历
- **中**：中组发〔2019〕21 号《公务员调任规定》明确调任对象为"调入机关担任领导职务或四级调研员以上"，学历审查存在国民教育序列要求
- **长**：中组发〔2019〕21 号《公务员调任规定》（中央组织部 2008 年制定，2019-10-15 修订，2019-11-26 施行），适用于国有企业、高等院校和科研院所等不参照公务员法管理的事业单位人员调入机关担任领导职务或四级调研员以上职级；调入中央/省级机关要求大学本科以上、调入市/地级以下机关要求大学专科以上；专业技术人员调入需担任副高级专业技术职务 2 年以上或正高级。实操中对学历的"国民教育序列"属性有审查要求，党校单证在此类资格审核中存在被认可程度差异（具体执行视调入机关和岗位规定）。

### PD-06 · 深化职称制度改革

- **文号**：人社部 2017 年《关于深化职称制度改革的意见》+ 各省 2024-2026 配套
- **短**：职称评审学历是重要资格
- **中**：人社部 2017 年深化职称改革意见 + 各省 2024-2026 配套，中高级职称评审中学历是重要资格条件，部分系列对学科对口度有要求
- **长**：人社部 2017 年《关于深化职称制度改革的意见》+ 各省 2024-2026 配套文件，中高级职称评审中学历是重要资格条件；部分系列要求国民教育序列研究生学历，部分系列对学科专业对口度有要求（如工程、医学、教育系列）。党校单证在专技岗职称评审中存在地区 / 单位认可度差异——体制内行政管理类岗位认可度较高，专业技术类岗位认可度较低（具体执行视本地评审细则）。

### PD-07 · 2026 国考职位表

- **文号**：中央组织部 / 国家公务员局 2026 年度国考职位表
- **短**：国考限研岗 7% → 12.95%
- **中**：2022 → 2025 国考研岗占比从 7% 上升到 12.95%，**近 60 个国考岗位**从"本科可报"变"仅限研究生"
- **长**：中央组织部 / 国家公务员局 2026 年度国考职位表，报考年龄上限放宽至 38 岁（应届硕博放宽至 43 岁）；研岗占比 7%（2022）→ 12.95%（2025）；**近 60 个国考岗位**从"本科可报"变"仅限研究生"，涉及银行、监管、海事、边检等核心系统。

### PD-08 · 末等退出 + 央国企薪酬改革

- **文号**：国企改办发〔2021〕7 号《关于加大力度推行经理层成员任期制和契约化管理有关事项的通知》+ 国务院国资委 2024-09-27 专题推进会部署 + 2024-12 国资委苟坪解读 + 2025/12 中央企业负责人会议（2026 五大重点任务含"深化三项制度改革"）
- **短**：国企末等退出常态化
- **中**：国企改办发〔2021〕7 号确立"末等调整和不胜任退出"制度，国资委 2024-09 会议深化部署，2025 年央企必须普遍推行（适用对象：管理人员，非普通员工）；央国企薪酬与绩效约束持续强化
- **长**：国企改办发〔2021〕7 号《关于加大力度推行经理层成员任期制和契约化管理有关事项的通知》明文写入「**末等调整和不胜任退出**」，作为"劳动、人事、分配"三项制度改革的"牛鼻子"。2024-09-27 国务院国资委专题推进会深化部署，明确**适用对象为管理人员，非普通员工**，"末等"不能简单划比例、不能搞"一刀切"。2024 年央企二三级子企业覆盖面不低于 70%，2025 年央企必须普遍推行。2025/12 中央企业负责人会议明确 2026 五大重点任务含"深化三项制度改革，持续提升经理层成员任期制契约化管理水平"——经理层任期制契约化、末等调整、不胜任退出、绩效刚性兑现等约束持续强化，核心导向是显性条件、量化考核（学历作为少数可量化、可主动获取的显性条件之一，价值进一步凸显；具体执行视个人单位情况）。

> ⚠️ **降级说明（2026-05-28）**：原"2026/01 国务院国资委 + 人社部 + 财政部三部委联合发布全国央国企薪酬管理新规"表述未核到原发文件，已降级为"薪酬与绩效约束持续强化"，依据 M1 §七 / M2 §C.4 / M3 §六.1 / 事实审计 C005。
- ⚠️ **历史演变备注**（仅供内部参考，不对外宣讲）：2021 年 7 号文原文表述为"**末位调整**"，2024-09 国资委深化部署后官方口径明确为"**末等调整**"——研知道统一用"**末等调整**"口径（贴最新官方表述）。

### PD-09 · 延迟退休 ⭐ 政策事实 vs 派生判断必须区分

- **文号**：全国人大常委会 2024-09-13 通过《关于实施渐进式延迟法定退休年龄的决定》+《国务院关于渐进式延迟法定退休年龄的办法》
- **短**：延迟退休改革落地
- **中（政策事实层）**：2024-09-13 全国人大常委会通过渐进式延迟退休决定，2025-01-01 起用 15 年时间逐步实施，男 60→63 / 女 50→55 或 55→58（延 3-5 年）
- **长（政策事实层 + 派生判断层）**：全国人大常委会 2024-09-13 通过《关于实施渐进式延迟法定退休年龄的决定》，2025-01-01 起用 15 年时间逐步实施。**政策事实**：男职工 60→63 岁（延 3 年），女职工 50→55 岁（延 5 年）或 55→58 岁（延 3 年）；坚持小步调整、弹性实施、分类推进、统筹兼顾原则。**研知道派生判断**：体制内职业生涯对应延长 3-5 年，学历投资回收期同步放大 3-5 年的窗口期空间（基于政策事实的判断方向，不是政策原文）。

⚠️ **宣讲口径区分**（关键）：

| 层 | 表述 | 是否政策原文 |
|---|---|---|
| 政策事实 | 延迟退休改革启动 / 延迟 3-5 年 / 15 年逐步实施 / 男 63、女 55 或 58 | ✅ 有原文支撑 |
| 研知道判断 | 3-5 年窗口期 / 体制内职业生涯延长 3-5 年 / 学历投资回收期放大 3-5 年 | ⚠️ 派生判断，与政策事实同步口径 |

**两者不能混淆**：

- 前者是政策事实（全国人大常委会决定原文）
- 后者是研知道基于政策的趋势判断
- 对女性体制内用户：不说死具体年龄数，统一用"延迟退休改革落地"+"3-5 年（视性别和原退休年龄）"避免性别歧义

## 四、改动触发流程

政策事实层有变化时按以下流程：

1. **先改本表**：调整对应 PD 的"中 / 长"档（短档相对稳定）
2. **全局 grep 旧表述**：在选校工具 + L1 两层做扫描，列出残留位置
3. **批量对账下游**：
   - 选校工具 docs（PRD / 06 推荐逻辑 / 数据来源映射 / 内容审核 / 小程序策划）
   - 选校工具 data（policy_basis.json / understand_long_form.json / rules.json / copy.json / zexiao.json）
   - 选校工具原型 HTML（按 ww 安排决定是否同步）
   - L1（M1-M9 对应段落 + 价值观层"正例"段）
   - 对外内容（小程序、推文、销售脚本等）
4. **L1 §六 交叉验证**：确认与 M3 §六 政策原文引用一致
5. **改动记录**：本表底部追加改动 changelog

## 六、ER2 元规则约束

所有"中 / 长"表述末尾必须遵循以下四条：

1. **用"可能性 / 判断方向"措辞**，不用"一定会 / 必然 / 保证 / 100%"等确定句式
2. **涉及"具体执行"留口子**：尾部加"具体执行视个人单位情况" / "（基于核实政策事实的可能性和判断方向）"
3. **附文号 + 发布部门 + 日期**：政策段必须可追溯到原文档
4. **文号必须能在一手官方信源独立验证**：
   - 一手官方信源 = `gov.cn` / `12371.cn` / 各级中央部委官网 / 各省政府门户 / 党中央机关报（人民日报、新华社）
   - **不算一手信源**：WebSearch 结果中 AI 总结/拼接的"以上规范主要依据 XXX"这种语句、知乎/百度百科/自媒体引用、未注明来源的 PDF 截图、培训机构的"政策解读"
   - **验证流程**：找到文号 → 必须打开至少 1 个官方域名页面看到文号原文 → 否则降级处理（不列文号，或加"待核实"标注）

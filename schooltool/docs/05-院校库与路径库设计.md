本文件解决两个问题：

1. **V1 到底先录哪些学校和路径**

## 二、V1 内容库拆分

V1 先拆成两层：

### 2.1 路径库

先回答"有什么路可走"。

**V1 业务边界（ww 2026-05-19 裁决封板）**：研知道只提供 **2 条主路径**：

1. **川渝党校**（=党校在职研究生 / 路径 A · 学信网不可查 · 体制内认可）—— 含四川党校 + 重庆党校
2. **非全管综 MPA / MBA / MEM**（=全国统考非全日制 / 路径 B · 学信网可查 · 学位+学历双证）—— 含外省其他党校 MPA、川渝及外地 985/211/普通院校 MPA·MBA

**不在 V1 路径库内**（业务边界外，05 不体现）：

- 合作办学（一年制硕士 / 留服认证）—— ww 决议：不做此业务
- 同等学力申硕（仅学位证）—— ww 决议：不做此业务
- 教育硕士、法学相关非全 等学位类型 —— 归并到"非全管综"路径下作为方向/院校项目选择，不作独立路径
- 全日制研究生 —— 在职人群范围外

**特殊场景处理**：

- 大专客户 —— 党校阻断（M7 R3 第一段），满 5 年工龄走非全管综接受大专项目（M9 §17：26% 项目）；**"先升本再读研"不作独立路径**
- 英数短板客户 —— R13 反向追问流程（六附.5），不作独立路径
- 硬阻断结果（年龄、身份硬不符等）—— 结果页"暂不建议立即报考"作为**输出状态**，不作独立路径

### 2.2 院校项目库

再回答"具体可报哪些项目"。

院校项目库不追求一开始全量，而按咨询频次和结果页优先级覆盖。

## 三、V1 首批覆盖原则

### 3.1 按咨询频次优先

先覆盖顾问最常被问到的学校和路径，而不是先追求“大而全”。

### 3.2 按结果页优先

结果页会被推荐到的项目，必须优先入库并核验。

### 3.3 按页面曝光优先

首页、了解页、结果页、院校详情页会直接展示的内容，优先核验。

### 3.4 按风险高低优先

以下信息优先核：

- 报考条件
- 是否需要党员
- 是否认可用于特定目标
- 学费与学制
- 特殊限制条件

### 3.5 元规则约束（来自 M7 §7.0）

所有路径推荐、风险提示、结果页文案在落地前必须先满足三条元规则——这是 M7 选校判断 skill 在 2026-05-18 入库的强约束：

- **R10 不打"主力客群"标签**：推荐文案是"场景级"——对遇到该场景的客户全部适用，不做"哪类客户最多/谁是主力/谁最强"概率判断。主力客群描述以客户管理客户资料的清单分布为依据
- **R11 "可能性和方向"口径**：所有路径推荐与风险提示文案，统一用"基于核实政策事实的可能性和判断方向"措辞，不做确定的因果论断
- **R12 三个硬约束**：不催单、不在公开渠道施压、不让客户朋友圈尴尬（合规三线：党纪政纪 / 不犯错 / 圈子重合）

## 四附、路径事实层（M3 全量入库 · baseline 2026-05-19）

> 本节为 M3-五种在职读研路径对比 全量事实层入库。**M3 是 5 种路径事实层；05 V1 路径库压缩为 2 条业务主路径（v0.4 ww 裁决）**——其中 M3 "合作办学 / 同等学力申硕"事实保留，但研知道不做此业务（C3/C4）。详见 §十一。

### 四附.1 五种路径总览表（来源 M3 §一）

| 路径 | 证书 | 入学难度 | 毕业难度 | 典型费用 | 学制 |
|------|------|----------|----------|----------|------|
| **全日制研究生** | 双证（学历+学位） | 高 | 高 | 1.5-30 万 | 2-3 年 |
| **非全日制研究生**（MPA/MBA/MEM 等） | 双证（学历+学位） | 中高 | 中高 | 5-50 万 | 2.5-3 年 |
| **合作办学** | 学位证 + 留服认证 | 中高 | 中低（2026 新规：境外≥180 天硬约束） | 15-40 万 | 1.5-2 年 |
| **党校在职研究生** | 仅学历证 | 中低 | 低 | 低至万元级 | 2-3 年 |
| **同等学力申硕** | 仅学位证 | 低 | 高 | 3-8 万 | 3-5 年 |

**定位一句话**：非全双证在难度和含金量上居中，是在职人群需要"硬通货"时的最优平衡点；党校在职入学有一定门槛但毕业容易，适合时间精力有限、目标是本单位内部竞争不被减分的人群。

**2026 政策强化信号**：教育部 2026 全国硕士研究生招生工作管理规定（2025-09-18 印发）明确"全日制和非全日制执行相同政策和标准""非全日制原则上招在职定向就业""全日制与非全日制招生计划不得相互调整使用"——非全双证的政策合法性进一步固化。

### 四附.2 党校在职研究生 vs 国民教育硕士（199 联考）对比（来源 M3 §二）

**范围说明**：本表仅讨论硕士层级。中央党校博士只招全日制非定向就业（基本学习年限 4 年），与"在职研究生"路径不重叠；省委党校非全日制硕士才面向在职定向就业人员。在职人员想读党校学历，实际落点是省委党校或市级党校的在职研究生班。

| 对比维度 | 党校在职研究生 | 国民教育硕士（199 联考） |
|---------|-------------|---------------------|
| 学信网可查 | 否 | 是 |
| 遴选报名 | 部分不认可 | 完全认可 |
| 干部提拔使用 | 认可 | 认可 |
| 学位授予 | 无学位 | 有硕士学位 |
| 调任资格 | 存在争议 | 完全认可 |
| 职称评审 | 部分认可 | 完全认可 |
| 社会认可度 | 体制内认可，体制外不认可 | 全社会认可 |

**关键事实边界**：

- 党校学历在遴选报名中，多数省份要求"国民教育序列"学历 → 省级/市级党校在职研究生通常不符合报名条件
- 党校学历在干部提拔使用中有效 → 本单位内部晋升、组织部考察认可
- 国民教育硕士（MBA/MPA/MEM）在所有场景中都被认可，包括遴选/职称/调任

### 四附.3 五种路径适用场景（来源 M3 §三）

| 路径      | 适合人群（客观条件）                                 | 不适合人群                                            |
| ------- | ------------------------------------------ | ------------------------------------------------ |
| 全日制研究生  | 能脱产 2-3 年；年龄 35 以下优先                       | 在职人员（体制内请假硬约束）                                   |
| 非全日制研究生 | 在职 + 需要双证硬通货；能腾出固定时间上课                     | 时间极度有限；预算有限                                      |
| 合作办学    | 预算充足；看重国际背景；对学信网可查不强求；**能满足境外≥180 天的时间约束** | 预算有限；目标是体制内遴选；**无法出境或无法满足境外天数（体制内请假难、岗位绑定强的人群）** |
| 党校在职研究生 | 在职 + 本单位深耕；时间精力有限                          | 需要遴选/调任/职称评审的场景                                  |
| 同等学力申硕  | 在职 + 需要学位证但不急；能投入长期备考                      | 需要学历证的场景；需要快速拿证                                  |

### 四附.4 路径选择的关键事实（来源 M3 §四）

- **双证中"非全 vs 合作办学"的区别**：非全属于国民教育序列（学信网可查），合作办学走留服认证（教育部留学服务中心认证）。两者在体制内认可度的关键差异在"是否学信网可查"
- **党校学历的"时间"特殊性**：中央党校全日制有国民教育资质（一般认可）；省级党校在职研究生通常不在国民教育序列；市级党校学历一般不认可
- **同等学力 vs 非全**：同等学力是"拿学位不拿学历"；非全是"学历学位双证"。对要求"硕士研究生学历"的场景，同等学力不顶用
- **合作办学的"境外天数硬约束"（2026 新规）**：一年制硕士境外实际学习≥6 个月（部分专业≥8 个月），中外合作办学项目境外累计≥180 天。系统自动比对申请人近 10 年出入境记录，"刷签证"或纯线上学习已无法通过留服认证；强制实人认证（国家政务平台 / 移民局 12367 APP 人脸核验）；造假后果为直接不予认证 + 黑名单 + 不退费 + 不可复核。这条直接砍掉"完全在国内拿留服认证"的灰色操作，对体制内请假难、岗位绑定强的人群是硬约束。

## 五、V1 首批院校项目范围建议

### 5.1 必须先完整核验的项目（川渝党校 · 路径 A）

- 四川党校在职研究生全部 V1 展示项目（3 个方向：经济学 / 政治学 / 法学）
- 重庆党校在职研究生全部 V1 展示项目（5 个方向：经管 / 法律 / 公管 / 党政 / 战略）
- **路径 A vs 路径 B 同名同所对照**——四川党校在职研究生（学信网不可查，学费 7200/年 ×3）vs 中共四川省委党校 MPA（学信网可查，学费 2.4 万 / 分数线 213）；重庆党校在职研究生 vs 中共重庆市委党校 MPA（学费 2.7-3.6 万 / 分数线 206）。这是 M9 §2.2 / §5.3 / §19.2 强调的"用户认知最大坑"，列表页与详情页必须清晰区分体系归属

### 5.2 必须先完整核验的项目（非全管综 · 路径 B · 川渝聚焦）

- **川渝党校 MPA 2 个**（中共四川省委党校 MPA / 中共重庆市委党校 MPA · 反常识极值必收）

⚠️ 本设计文档必须区分两个独立的考研体系：

### 五附.7 路径 A 党校在职研究生（来源 M9 §4）

**路径本质**（M9 §4.1）：

- 办学主体：省/市委党校（不归教育部管辖）
- 考试机构：党校自办自考，不参加全国硕士统考
- 证书：盖省/市委组织部 + 党校印章的研究生毕业证书；**无学位证**
- 学信网：不可查（非国民教育序列）
- 政策依据：中央 [2000] 10 号文件——党校学历「待遇参照国民教育相应学历的有关规定」
- 实际认可：体制内（公务员遴选、干部晋升、国央企内部）普遍认可；私企/教育部口径场景不可用
- 核心人群：体制内在职干部、想以低成本拿到体制内认可学历的人

**党校在职研究生对体制内人群的三个不可替代**（M9 §4.1·补）：

| 维度 | 不可替代点 |
|---|---|
| **师资** | 党校老师不仅是学者，更是体制内政策制定、解读、培训的核心力量。讲党建、讲政策、讲基层治理，没有比党校更对口的师资 |
| **同学** | 党校同学几乎全部来自体制内（各级党政机关、事业单位、国央企），学习语境天然对齐，不存在 MBA/MPA 同学圈层落差大的问题 |
| **培养路径** | 党校研究生培养目标明确——「干部 + 治理」路径。如果你的职业方向是体制内晋升、地方治理、党务/组织/纪检/统战，党校的培养体系比任何 MPA 都更贴合实际工作场景 |

**2026 教师编制动态调整与县域统筹对党校价值的放大方向**（基于核实政策事实的可能性）：教师编制动态调整、县域统筹、跨校调配与岗位流动机制继续推进 → 跨校流动与岗位竞争被推紧的政策方向下 → 党校"全省体制内同学网络"（含教育系统）对教师跨校协调、政策理解、职称评审年限优势有对应政策依据。

⚠️ 教师群体在党校招生中的实际占比变化需后续招生数据验证。

**"不考英数"是优势还是陷阱**（M9 §4.1·注）：

| 指标 | 党校在职研究生 | 双证 MPA |
|---|---|---|
| 是否考英数 | 不考 | 考英语二 + 管综 |
| **自然过考率** | **不到 10%** | **20% 以上** |
| 题型 | 全主观题（写作 + 论述） | 选择 + 写作 |
| 真正考察 | 政治理论 + 专业理解 + **结构化书面表达** | 应试技巧 + 基础学科能力 |

**抛开"英数恐惧"假象**，党校在职研究生反而比双证 MPA 更难考。核心难点：全主观题（无猜对可能）/ 政治理论 + 专业基础（需系统理解）/ 论述题要求结构化表达。

⚠️ **选题警示**：把党校包装成"低门槛省力学历"是错误方向。研知道在做党校相关内容时，应强调"更对口"而非"更省力"。

**四川党校 vs 重庆党校核心对比**（M9 §4.2）：

两校共同特征：都不考英数 / 都是党校自办自考、学信网不可查 / 都是 3 年学制 / 都对体制内人群有高认可度。

### 五附.8 路径 B MPA / MBA 非全日制（来源 M9 §5）

**路径本质**（M9 §5.1）：

- 办学主体：教育部认可的高等院校（含部分省/市委党校的 MPA 项目）
- 考试机构：全国硕士研究生统一招生考试（管综 / 综合能力 + 英语二）
- 证书：硕士研究生学历证书 + 硕士学位证书（**学历 + 学位双证**）
- 学信网：✅ 可查
- 核心人群：体制内 + 体制外都覆盖；想要学信网可查学历的人

**MPA 价值定位**（M9 §5.1·补 · 用于内容创作）：

一句话：MPA 不是一张学历，是**"进入中层治理结构的通行证"**。如果 MBA 培养的是"懂管理的企业经营者"，MPA 就是"懂管理的政治官僚"。

**适合岗位**：街道办、医院行政、学校管理、财政、人社、规划、项目办、应急、民政、社区治理——只要工作场景与"政策落地 + 跨部门协作 + 治理执行"相关。

MPA 五门核心课程对照：

| 课程 | 学什么 | 实际工作帮助 | 内容钩子 |
|---|---|---|---|
| 公共政策理论与实践 | 政策怎么诞生、博弈、推进；中央到地方的传导逻辑 | 写材料/做方案时**说得明白讲得动人**；对政策窗口期更敏感 | "政策窗口期" / "材料感" |
| 公共政策分析 | 怎么判断该不该用政策解决；怎么选政策工具 | 提升结构化写作 + 逻辑型表达；独立策划完整执行链 | "结构化表达" |
| 中国式现代化治理 | 中国为什么走这条发展路线；治理现代化与岗位的关系 | 把"国家战略"翻译成"地方操作"；**在大政方针下找到自己的位置** | "基层治理" / "试点" |
| 政府绩效管理 | 工作目标如何量化拆解；让上级信服、同级配合、下属执行 | 知道**领导和组织实际怎么考核观察潜在提拔对象** | "组织视角" |
| 公务员制度与人事行政 | 干部怎么选拔；事业 vs 公务员差异；人事博弈 | 从制度设计者视角理解职业路径，**破除"只看领导喜好"误区** | "晋升逻辑" |

**MPA 想补的 4 种能力**：政策理解力 / 结构化表达 + 公共沟通 / 跨部门协同的组织视角 / "问题—方案—执行—评估"的逻辑闭环。

**研知道真实观察**：北大 MPA 圈基层多、中高层少；会实干的多、会提炼的少。MPA 补的就是"提炼能力"和"治理视角"。

**2026 政策环境对 MPA 价值的放大方向**（M9 §5.1·补）：

- 延迟退休配套政策 → 体制内职业生涯有延长 3-5 年的政策空间 → MPA"通行证"作用周期对应延长，长期 ROI 提升的政策依据增强
- 国企末等退出常态化（2024 央企 6%）→ 国企中层"稳定"逻辑面临政策环境变化 → MPA + 调任成为政策支持的可能路径

⚠️ 调任年龄上限（按职级）、调任窗口（副处/副高以上才有资格）等具体执行视个人情况评估。

**MBA 价值定位**（M9 §5.1·补 · 用于内容创作）：

一句话：MBA 不是学"术"，是**换"脑"**。在职读 MBA，是在实践中升级自己的最佳路径之一。

**对人脉论的态度**：10 年前的 MBA 确实是"学历 + 财富筛选 + 人脉机器"；今天的 MBA 在人脉维度上"贬值"了，但在"换脑"维度上反而更值钱。研知道真实观察：MBA 中层多、大佬少（但小老板不少）；认真上课的少、上升期的人多、上位者少。**冲着人脉混圈子的内容口径已经落伍**。

MBA 五门核心课程：

| 课程 | 解决什么 | 内容钩子 |
|---|---|---|
| 战略管理 | **定方向**——懂战略的人不焦虑，他知道"慢慢来也是对的方向" | "战略思考" / "方向焦虑" |
| 营销管理 | **创造并传递价值**——营销自己是成为组织"恰好需要的那个人" | "个人营销" |
| 人力资源管理 | **搞人 / 组团队 / 带队伍**——一个人再能干，也干不过一个团队 | "影响力" |
| 运营管理 | **提效率**——解决流程、制度、岗位协作 | "效率破局" |
| 财务管理 | **理解「钱」的语言**——风险管理和资源配置的语言 | "财务思维" |

**院校间差异比 MBA 和 MPA 之间还大**——读哪个 MBA，最好从这五门课的具体设计开始判断，不要看机构推荐和小红书。**这是研知道内容差异化的核心切入点：带用户去看官网培养方案**。

**2026 政策环境对 MBA "换脑"价值的放大方向**：央企末等退出 + 薪酬管理新规下，部分央企子公司负责人年薪从 491 万下调至 241 万 → "MBA = 人脉圈子"作为唯一价值的旧逻辑面临政策环境变化 → "5 门课换脑"的真本事价值在政策环境下相对增强。

### 五附.11 给五类人的路径建议（来源 院校库认知框架 §十）

- **公务员 / 事业编（含党员）**：四川党校在职研（首推，川人）/ 重庆党校在职研（首推，渝人）/ 川大或电子科大或重大 MPA（双证补充）/ 川渝党校 MPA（207-213 分可冲）
- **国央企在职**：MPA 优先（含调任路径预留）/ 党校在职研（如果不需要双证）/ MBA（如果有市场化竞争和创业打算 · 2026 政策环境新增）
- **民企 / 私营在职（无党员身份）**：MBA（首选）/ MPA（双证选择）/ ⚠️ 不推荐党校在职研（私企场景认可度存疑）
- **大专学历 / 工作经验 5 年+**：双轨——A 接受大专的 MPA 项目（26% / 5 年工龄）直接报；B 先升本再读研（教育系统传统路径）。**05 V1 当前只接 B 轨**
- **国企 / 央企 34-36 岁中层**（2026 政策环境新增覆盖 · 潜在场景而非主力客群）：M7 §7.1 R8 双线评估推荐：X1 MBA "换脑" 应对市场化竞争（重大 15.6 万 / 西南交大 10-13 万 / 西南财经 11-16 万）；X2 MPA + 调任公务员转编（重大 9.6 万 / 西南交大 8.4 万 / 重庆党校 MPA 2.4 万 213 分）

### 五附.12 核心结论（来源 M9 §六部分）

1. **体制内考研其实是两条独立路径**：路径 A（党校自办、不可查、体制内认可）vs 路径 B（统考、可查、全社会认可）
2. **党校"双线并行"是用户认知最大坑**：四川/重庆党校同时开了两个独立项目，学费几乎相同（2 万 vs 2.4 万），但证书性质完全不同
3. **MBA 和 MPA 是两种完全不同的游戏**：MBA 花钱买学校（80% 分数线一致），MPA 考分选学校（70% 学费集中）
4. **分数线四年连降**：MBA 170→151（降 19 分），MPA 178→164（降 14 分）。现在是近 4 年最容易考的时候
5. **B 线不是「便宜线」而是「分数线」**：B 线学费 ≈ A 线，价值纯粹在于分数线低 10 分
6. **川渝市场的特殊性**：川渝 MBA 全部 = 151 分（无分化）/ 川渝 MPA 分化大（164-213 分）/ 川渝 985 比全国 985 便宜 40% / 川渝无提前面试 / 川渝党校 MPA 是全国 MPA 反常识极值

## 六附、选校规则与应答框架（M7 全量入库 · v0.1 baseline 2026-05-18）

> 本节为 M7-选校判断逻辑 §七 全量入库——12 条规则集 + 1 个应答框架。每条规则均有 ≥3 处独立证据来源（M1-M9 + 价值观层 + 选校工具 06）。

### 六附.1 元规则（强约束 · 所有规则必须先满足）（来源 M7 §7.0）

- **R10 不打"主力客群"标签**：M7 规则一律是"场景级"——对遇到该场景的客户全部适用。不做"哪类客户最多/谁是主力/谁最强"概率判断。主力客群描述以客户管理客户资料的清单分布为依据（依据：经营战略总纲 §5.2 公理第 ⑥ 类 / M4 §八 / 认知框架 §十.5 注）
- **R11 推荐文案用"政策支持的可能性和方向"口径**：所有路径推荐与风险提示文案，统一用"基于核实政策事实的可能性和判断方向"措辞，**不做确定的因果论断**（依据：上下文价值观层 §3.8 C2 / M9 §5.1·补 / §4.1·补）
- **R12 必须尊重三个硬约束**：所有推荐流程不催单、不公开渠道压力、不让客户朋友圈尴尬。三个硬约束 = 党纪政纪 / 不犯错 / 圈子重合（依据：M5 §五.2 / 业务必读 §5.1 / 内容生产手册 §一.3 合规三线）

### 六附.2 路径分叉规则 5 条（来源 M7 §7.1）

- **R1 体制内党员 → 路径 A 优先**：客户身份=体制内（公务员/事业编/国央企）+ 党员身份 → 首推路径 A（四川/重庆党校在职研究生）。判据：体制内认可度 + 性价比 + 起点级别价值在县乡场景已大部分锁定（依据：认知框架 §十.1 / 选校工具 06 P001 / M3-学历作用 §1.4）
- **R2 民企无党员 → 路径 B（重庆党校仅作慎选）**：客户身份=民企/私企 + 无党员身份 → 排除四川党校（不接受非党员）；重庆党校虽然不限党员但**私企场景认可度存疑慎选**，主推路径 B（MPA/MBA）（依据：认知框架 §十.3 / 选校工具 06 E001 / M9 §4.4）
- **R4 遴选/调任/职称 → 双证（MPA/MBA/MEM）**：客户目的属"遴选报名 / 调任公务员 / 职称评审年限优势" → 必须国民教育序列双证。党校在职研究生在这些场景中存在认可度争议，风险不能承担（依据：M3 §1.3/§1.5/§1.6 / 选校工具 06 P002 / 认知框架 §九 Q9）
- **R5 教育/医护专技岗 + 主诉职称 → 降低党校优先级**：客户身份=教师/医护事业编专技岗 + 主诉"想要职称评审年限缩短" → 降低党校优先级，推 MPA / 学科对口硕士。教育/医护专技岗有刚性双证需求（依据：M3 §1.5 / 选校工具 06 E002 / M9 §4.1·补）
- **R8 国企 34-36 中层 → X1 MBA 换脑 / X2 MPA+调任 双线**：客户身份=国企/央企 34-36 岁中层 + 末等退出风险敞口高 → 双线评估推荐：X1 MBA "换脑" 应对市场化竞争（重大 15.6 万 / 西南交大 10-13 万 / 西南财经 11-16 万）；X2 MPA + 调任公务员转编（重大 9.6 万 / 西南交大 8.4 万 / 重庆党校 MPA 2.4 万 213 分）。⚠️ 推荐时必须按 R10 不打"主力客群"标签——本规则对遇到该场景的客户适用，但不代表此画像是研知道主力客群（依据：认知框架 §十.5 / M9 §5.1·补 / M1 §七+M2 §C.4）

### 六附.3 阻断规则 1 条（来源 M7 §7.2）

- **R3 大专 → 阻断路径 A**：客户当前学历=大专 → 阻断路径 A（四川党校不接受大专；重庆党校 2020 起不接受专科同等学力）。转路径 B（**26% 的 MPA/MBA 项目明确接受大专报考**，需满 5 年工作经验）；**或先升本承接**（依据：认知框架 §十.4 / 选校工具 06 B001 / M9 §4.4 / M9 §17）

⚠️ R3 是**双轨阻断**——"转路径 B 接受大专项目" + "先升本承接"并列，05 V1 现状只接了后者（见 §十一 C5）。

### 六附.4 风险提示规则 3 条（来源 M7 §7.3）

- **R6 党校"不考英数"≠容易考**：客户主因是"不考英数" → 提示党校自然过考率<10%（双证 MPA 20%+），考察方式换成"全主观题 + 结构化书面表达"，对非应试型考生未必更友好（依据：M9 §4.1·注 / 认知框架 §九 Q9 / M9 §4.2）
- **R7 川渝党校 MPA 反常识极值（最便宜+最高分 206-213）**：客户主因是"低学费"想推党校 MPA → 提示川渝党校 MPA 分数线 206-213（全国 MPA 最高梯队），不是省力路径（依据：M9 §四 / 认知框架 §1.2 坑3 / 认知框架 §九 Q4）
- **R9 35→38 双轨叙事**：客户年龄段在 35-38 之间 → 必须用双轨提示（**政策红线 38 / 实操红线 35**），不能单轨说"35 还有机会"或"38 没机会"。具体应用：国考限研究生 / 川渝公考遴选 / 末等退出（依据：M2 §E.0 / M4 §一 / M4 §五）

### 六附.5 R13 反向追问应答框架（来源 M7 §7.4 · v0.2 校准 2026-05-18 PRD v3.1 评审）

**R13 英数短板触发反向追问框架（不直接 → 党校）**：客户主诉"英语不行 / 数学不行 / 怕英数 / 高中就不及格"等表达 → **不直接给路径建议**，先跑三层（含二级追问最多四步）反向追问：

**① 能力真伪辨**："是真实测试过还是恐惧？"
- 测试过 → 看具体分数 / 距门槛差距
- 没测试 → 透出门槛信息：**非全管综英语二满分 100 只需过 34-38 分**；管综数学含基础高中+大学初级，**不是考研数学一/二难度**

⚠️ 此数字目前未入 M9 / 认知框架，下次跑院校库 skill 需补（M7 §九 下游对账行 high）。

**② 动机和需求辨**："你的目标场景对学位类型有要求吗？"
- 必须双证（遴选/调任/职称硬要求）→ 即使英数恐惧也得过，门槛比想象低
- 不刚（仅内部竞争防御）→ 党校"不考英数"才是真省力点
- 先有证再优化（防御性立场）→ 推党校优先 + 双证备选

**③ 能力偏好辨**："你更倾向哪种思维方式？"
- 理科思维（逻辑推理强、对数字符号敏感）→ 双证管综+英二其实更对路
- 文科思维（文字表达强、阅读理解强）→ 党校全主观题更对路
- 说不准 → 进 ③.b 二级追问

**③.b 努力偏好辨**（仅 ③ 选"说不准"看到）："你更愿意在理解上努力，还是在大量背诵上努力？"
- 理解努力 → 类管综（P002）
- 背诵努力 → 类党校（P001）

**追问全部跑完后**才给路径建议。直接 "英语短板→党校" 会把双证用户错配到党校（自然过考率<10%），可能比双证更难考。

⚠️ R13 与 05 §四 "暂不建议立即报考"路径设计的边界需厘清——见 §十一 C6。

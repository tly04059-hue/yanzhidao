# 选校诊断工具 · 推荐逻辑规则表

> 版本：v0.1 | 日期：2026-05-05
> 用途：把顾问经验、推荐逻辑、阻断逻辑、风险提示从 prose 变成结构化规则
> 前置：`专项/01-诊断逻辑与推荐引擎.md`、`04-数据字典.md`

---

## 一、文档目的

本文件是推荐系统的执行层文档。

它不替代推荐逻辑说明文档，而是把“解释性内容”进一步拆成可落地的规则单元，供：

- 前端联调结果页
- 后端实现规则引擎
- 内容侧校对推荐理由
- 顾问回查特殊情况

---

## 二、V1 规则结构

V1 每条规则建议统一写成以下结构：

| 字段 | 说明 |
|------|------|
| rule_id | 规则编号 |
| rule_type | 规则类型：路径优先 / 排除 / 阻断 / 风险提示 / 备选补充 |
| condition | 命中条件 |
| priority | 优先级 |
| output_action | 输出动作 |
| output_copy | 输出文案 |
| fallback_action | 未命中时的兜底动作 |
| status | 启用 / 待确认 / 停用 |

---

## 三、V1 规则类型

### 3.1 路径优先规则

用于判断首推路径。

示例：

| rule_id | condition | output_action |
|---------|-----------|---------------|
| P001 | 目标=本单位晋升 且 对统考难度接受度低 | 首推四川党校或重庆党校 |
| P002 | 目标=遴选 / 调任 / 职称 且 需国民教育序列 | 首推 MPA 或其他双证路径 |

### 3.2 排除规则

用于把不适用路径排除。

示例：

| rule_id | condition | output_action |
|---------|-----------|---------------|
| E001 | 四川地区 且 非党员 | 排除四川党校 |
| E002 | 教育医疗专技岗 且 主要诉求是职称 | 降低党校优先级 |

### 3.3 阻断规则

用于直接结束常规推荐，改为承接路径。

示例：

| rule_id | condition | output_action |
|---------|-----------|---------------|
| B001 | 当前学历=大专 | 进入“先升本再读研”结果页 |
| B002 | 条件严重不匹配 且 期望不切实际 | 输出“暂不建议立即报考” |

### 3.4 风险提示规则

用于不改变主结果，但追加提示。

示例：

| rule_id | condition | output_action |
|---------|-----------|---------------|
| R001 | 重庆 + 遴选 + 年龄 > 35 | 提示年龄限制风险 |
| R002 | 法学方向 | 提示竞争激烈或通过率风险 |

### 3.5 备选路径规则

用于结果页生成第二、第三推荐路径。

## 五、推荐结果输出结构

推荐逻辑跑完后，结果页至少要拿到以下结构化输出：

1. 首推路径
2. 备选路径
3. 排除路径及原因
4. 风险提示
5. 推荐院校项目列表
6. 下一步建议

---

## 六、规则维护原则

1. 一条规则只解决一个判断。
2. 优先级冲突时，以阻断规则和排除规则优先。
3. 风险提示规则不直接改变主推荐，除非明确升级为排除或阻断。
4. 所有上线规则必须能回溯到：
   - 顾问经验
   - 历史案例
   - 官方政策
   - 已公开招生信息

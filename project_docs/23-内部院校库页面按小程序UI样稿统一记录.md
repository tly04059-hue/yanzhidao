# 内部院校库页面按小程序 UI 样稿统一记录

> 更新时间：2026-05-15  
> 修改范围：`schooldata/yanzhidao_schooldata.html`、`schooldata/yanzhidao_schooldata.css`、`schooldata/yanzhidao_schooldata.js`  
> 参考样稿：`schooltool/docs/ui_samples/05_院校列表页.html`、`08_院校详情页.html`  
> 说明：`schooltool/` 仅读取未修改。

## 1. 本次目标

把内部院校库 `yanzhidao_schooldata.html` 从原来的青绿色大屏风格，统一为小程序当前使用的研知道 v3 风格：

- 米色页面背景。
- 白色卡片。
- 橙色主强调。
- 绿色辅助状态。
- 圆角筛选 chip。
- 轻阴影和低饱和边框。
- 小程序一致的路径顺序：`MPA -> MEM -> MBA -> 党校`。

## 2. 已完成

1. 页面外壳重构：
   - 新增 sticky 顶栏。
   - 新增内部数据浏览标识。
   - Hero 改为 `School Library / 西南地区考研院校信息库`。
   - 增加数据分层说明：前台看核心字段，信息库保留完整来源和历史细节。

2. 筛选区调整：
   - 地区 chip：四川、重庆、云南、贵州。
   - 路径 chip：MPA、MEM、MBA、党校。
   - 路径展示顺序调整为 `MPA -> MEM -> MBA -> 党校`。

3. 院校卡片调整：
   - 白卡片 + 米色底。
   - 学校 Logo 改为柔和圆角头像样式。
   - 标签改为橙色/绿色胶囊样式。
   - 展开按钮改为橙色主按钮。
   - 展开后基础信息和拟录取分析保持原有数据结构。

4. 表格和党校数据样式调整：
   - 表头、边框、行 hover 统一为小程序风格。
   - 党校主要信息、招生专业与报录指标、院校分板块统一为白卡片展示。
   - 不改变 `data.json`、`dx_data.json`、`mem.json` 的字段结构和数据内容。

5. 缓存处理：
   - HTML 中 CSS/JS 引用增加版本参数，避免浏览器继续使用旧样式。

## 3. 验证结果

已验证：

```bash
node --check schooldata/yanzhidao_schooldata.js
python3 projectOps/scripts/check-all.py
```

浏览器验证：

- `http://localhost:8091/yanzhidao_schooldata.html`
- 首屏正常显示。
- MPA 默认列表正常显示。
- 点击学校“展开”后，基础信息和拟录取分析 tab 正常显示。
- 切换 MEM 正常。
- 切换党校正常，主要信息和院校分板块正常显示。

## 4. 注意事项

- 本次是样式统一，不是数据清洗。
- 信息库仍会展示比小程序更多的数据，这是符合“信息库数据大于前台展示数据”的规则。
- 后续如果继续优化该页面，可以再增加搜索、排序、字段完整度提示，但不要破坏现有 JSON 数据读取逻辑。

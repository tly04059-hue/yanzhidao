### 3. 前后端链路

当前提测版链路已打通：

```text
小程序/H5
  ↓
Node API
  ↓
schooldata/data.json
  ↓
推荐结果 / 院校列表 / 院校详情
  ↓
结果页
  ↓
留资写入 JSONL
```

### 4. 当前 API 服务

位置：

```text
schooltool/apps/api/server.js
```

本地默认地址：

```text
http://127.0.0.1:8010/api
```

已支持接口：

- `GET /api/health`
- `GET /api/miniapp/schools`
- `GET /api/miniapp/schools/:id`
- `POST /api/miniapp/recommend`
- `POST /api/miniapp/lead`
- `GET /api/miniapp/timeline/:pathType`
- `POST /api/miniapp/score-estimate`

### 5. 当前数据源

院校数据来自：

```text
schooldata/data.json
```

当前 API 可读取约 181 条院校项目数据。

### 6. 当前留资记录

提测版留资写入：

```text
schooltool/data/runtime/miniapp-leads.jsonl
```

后续正式上线建议迁移到阿里云 RDS MySQL。

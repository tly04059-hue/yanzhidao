# 阿里云与微信上线资源准备

> 目标：准备小程序上线所需的最小云资源、域名、HTTPS、小程序后台配置。

## 一、建议线上架构

```text
微信小程序
  ↓ HTTPS
https://yanzhidao.com/api
  ↓ Nginx 反向代理
Node API 服务
  ↓ 读取
schooldata/data.json
  ↓ 写入
runtime/miniapp-leads.jsonl（提测版）
```

正式稳定后再升级为：

```text
Node API
  ↓
阿里云 RDS MySQL
  ↓
后台管理系统
```

## 二、建议域名规划

| 用途 | 建议域名 | 必须程度 |
| --- | --- | --- |
| API 服务 | `yanzhidao.com/api` | 必须 |
| H5 入口 | `m.yanzhidao.com` | 建议 |
| 官网/介绍页 | `www.yanzhidao.com` | 可后置 |

小程序 request 合法域名必须是 HTTPS，且不能带端口。

## 三、需要你提供的信息

### 1. 阿里云账号与资源

请确认或提供：

- 阿里云账号是否已经有企业认证？
- 是否已有 ECS 服务器？
- 如果已有 ECS：
  - 公网 IP
  - 操作系统版本
  - 是否已安装 Node.js / Docker / Nginx
  - 是否开放 80、443 端口
- 如果没有 ECS：
  - 是否倾向 ECS
  - 是否倾向阿里云容器服务
  - 预算范围

### 2. 域名

请确认或提供：

- 是否已有域名？
- 域名是什么？
- 域名是否在阿里云管理？
- 域名是否已备案？
- 是否可以配置子域名：
  - `yanzhidao.com`
  - `m.yanzhidao.com`

### 3. SSL 证书

请确认：

- 是否已有 SSL 证书？
- 是否需要用阿里云免费证书？
- 证书要覆盖哪个域名？
  - 最少：`yanzhidao.com`
  - 建议：`yanzhidao.com`、`m.yanzhidao.com`

### 4. 微信小程序后台

请提供或确认：

- 小程序 AppID
- 小程序主体是否为企业主体
- 你是否有微信公众平台管理员权限
- 是否已经配置服务器域名
- 服务类目是否已确定
- 是否已有隐私协议/用户协议

## 四、阿里云最小部署步骤

### 1. 服务器准备

最小配置建议：

```text
1 核 2G / 2 核 2G 均可
系统：Ubuntu 22.04 或 Alibaba Cloud Linux
端口：80、443
```

### 2. 安装运行环境

需要：

- Node.js 20
- npm
- Nginx
- PM2 或 Docker

### 3. 上传项目

至少上传：

```text
schooltool/apps/api
schooltool/package.json
schooltool/package-lock.json
schooldata/data.json
```

### 4. 配置环境变量

生产环境建议：

```bash
NODE_ENV=production
HOST=0.0.0.0
PORT=8010
SCHOOL_DATA_PATH=/app/schooldata/data.json
RUNTIME_DIR=/app/runtime
CORS_ORIGIN=https://m.yanzhidao.com,https://www.yanzhidao.com
```

### 5. 启动 API

Node 方式：

```bash
cd schooltool
npm ci --omit=dev
npm run start:prod
```

Docker 方式：

```bash
docker build -f schooltool/apps/api/Dockerfile -t yanzhidao-miniapp-api:latest .
docker run -d --name yanzhidao-api -p 8010:8010 -v /app/runtime:/app/runtime yanzhidao-miniapp-api:latest
```

## 五、Nginx 反向代理

建议配置：

```nginx
server {
  listen 80;
  server_name yanzhidao.com;
  return 301 https://$host$request_uri;
}

server {
  listen 443 ssl http2;
  server_name yanzhidao.com;

  ssl_certificate /etc/nginx/ssl/yanzhidao.com.pem;
  ssl_certificate_key /etc/nginx/ssl/yanzhidao.com.key;

  location /api/ {
    proxy_pass http://127.0.0.1:8010/api/;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto https;
  }
}
```

验证：

```bash
curl https://yanzhidao.com/api/health
```

期望返回：

```json
{
  "ok": true,
  "service": "yanzhidao-miniapp-api"
}
```

## 六、小程序后台配置

微信公众平台：

```text
开发管理 → 开发设置 → 服务器域名
```

添加：

```text
request 合法域名：https://yanzhidao.com
```

注意：

- 必须 HTTPS
- 不能带端口
- 证书必须有效
- 域名需要备案
- 配置后重新上传体验版

## 七、前端生产构建

确认：

```text
miniapp/.env.production
VITE_API_BASE_URL=https://yanzhidao.com/api
```

构建：

```bash
cd miniapp
npm run build:mp-weixin
```

上传目录：

```text
miniapp/dist/build/mp-weixin
```

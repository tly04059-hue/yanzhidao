# 研知道项目协作资料

这个目录专门存放项目推进、提测、上线、云资源准备相关资料。

## 当前 P0 执行入口

生产/API/微信真机本地可检查项：

```bash
python3 projectOps/scripts/check-production-readiness.py
```

该脚本会输出 `projectOps/reports/latest-production-readiness-report.json`。其中 DNS、SSL 证书、微信后台合法域名、真机测试、销售承接方式仍需要人工确认。

## 当前原则

1. 每次开始更新前，先读 `00-项目更新前必读规则.md`。
2. 5 月 20 日前优先保证提测主链路跑通。
3. 页面内容可以先用模板数据，接口链路必须先跑通。
4. 数据源从 `schoolData/` 的发布层逐步替换旧数据，迁移前不破坏现有页面。
5. 留资提测版先写 JSONL，后续再迁移到 MySQL/RDS。
6. 后台管理系统后置，不阻塞小程序提测。
7. 跨目录数据更新、同步、校验、构建优先走 `projectOps/`，不要手工搬发布数据。

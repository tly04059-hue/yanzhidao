## 2. 重要规则

1. `school_code` 不硬猜；没有稳定来源时留空并标记 `needs_review`。
2. `province` 优先使用源表已有地区，其次使用 AIAgent 基础数据，再使用城市映射。
3. 多城市跨省或来源冲突时标记人工复核。
4. 同一学校补齐后，应回填到该学校所有 MPA/MBA 记录。
